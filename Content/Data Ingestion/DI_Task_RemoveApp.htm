Remove an application

 

This topic describes how to stop and remove instances of the Feed Acquisition, Message Shredder, and Data Parser applications.

To remove an application instance

  1. Stop the application service instance.

  2. Unregister the application instance using the following syntax:

    Unregister-AmInstanceWithLogicalHostInstance -InstanceType ApplicationInstance -InstanceName ApplicationInstanceName-LogicalHostName LogicalHostName

    For example, type the following cmdlet to unregister the Shredder_ADT_G instance from the Fabric logical host:

    Unregister-AmInstanceWithLogicalHostInstance -InstanceType ApplicationInstance -InstanceName Shredder_ADT_G -LogicalHostName FabricLogicalHost

  3. From the computer, cluster, or Network Load-Balanced (NLB) node where the application was deployed, type the following cmdlet:

    Sync-PhysicalHost

  4. Unpublish the application instance using the following syntax:

    Unpublish-AmApplicationInstance ApplicationInstanceName

    For example, type the following cmdlet to unpublish the Shredder_ADT_G instance:

    Unpublish-AmApplicationInstance Shredder_ADT_G
