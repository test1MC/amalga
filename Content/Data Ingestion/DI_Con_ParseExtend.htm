Add parser extensions

 

This topic describes two types of extension points for data parsers.

Transform library classes

A custom transform library is a class that includes methods that you can use for configuration-driven derivations. The process involves marking up a class and its methods with transform-related attributes directly in the service manifest.

For instructions and an example class, see Transform library class extension.

Custom assemblies

Unlike a transform library, a custom assembly can include complex types that derived from the IDataParserExtension interface. At runtime, the custom assembly executes within the instance.

For instructions and an example application, see Custom assembly parser extension.
