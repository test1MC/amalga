FeedInfo element

 

Declares a collection element for descriptive feed information.

Syntax

[ <Setting Name="FeedInfoXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <FeedInfo>
             <Name>...</Name>
             <Type>...</Type>
             <Provider>...</Provider>
             <TerminologyName>...</TerminologyName>
             <PreferenceRanking>...</PreferenceRanking>
             <Preferred>...</Preferred>
             <DefaultTimeZone>...</DefaultTimeZone>
          </FeedInfo>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements DefaultValue
Child elements DefaultTimeZone, Name, PreferenceRanking, PreferenceRanking, Provider, TerminologyName, Type

Remarks

The FeedInfo element is the root for FeedInfoXML setting. Always declare the FeedInfo element in the service manifest of the and applications. Do not declare this element in the manifests of other pipeline applications.
