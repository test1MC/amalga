ErrorDescription element

 

Declares a description of the exception to be used in the log entry.

Syntax

[ <Setting Name="ErrorHandlingConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <ErrorHandlingConfiguration>
             <ErrorHandlingPolicies>
                <ErrorHandlingPolicy>
                   <LogSetting>
                      <ErrorDescription>...</ErrorDescription>
                   </LogSetting>
                </ErrorHandlingPolicy>	
             </ErrorHandlingPolicies>
          </ErrorHandlingConfiguration>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements LogSetting
Child elements None

Remarks

Declare the ErrorDescription element only within a service manifest of the , , , and applications.

Example

<ErrorDescription>The component has attempted to connect to the database server; however, the connection has timed out.</ErrorDescription>
