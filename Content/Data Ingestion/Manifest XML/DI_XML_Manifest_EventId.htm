EventId element

 

Declares an event identifier to be used in the log entry.

SyntaxErrorTitle element

[ <Setting Name="ErrorHandlingConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <ErrorHandlingConfiguration>
             <ErrorHandlingPolicies>
                <ErrorHandlingPolicy>
                   <LogSetting>
                      <EventId>...</EventId>
                   </LogSetting>
                </ErrorHandlingPolicy>	
             </ErrorHandlingPolicies>
          </ErrorHandlingConfiguration>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length Integer
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements LogSetting
Child elements None

Remarks

The EventId element is mandatory to identify the event that throws the error. Avoid using the following ranges:

  • 10301-10900
  • 10901-11200
  • 10401-10500
  • 11201-11500
  • 11801-12100

Declare the EventId element only within a service manifest of the , , , and applications.

Example

<EventId>33001</EventId>
