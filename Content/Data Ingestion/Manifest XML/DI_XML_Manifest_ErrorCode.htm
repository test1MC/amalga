ErrorCode element

 

Declares the error code for an exception.

Syntax

[ <Setting Name="ErrorHandlingConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <ErrorHandlingConfiguration>
             <ErrorHandlingPolicies>
                <ErrorHandlingPolicy>
                   <ErrorCodes>
                      <ErrorCode>...</ErrorCode>
                   </ErrorCodes>
                </ErrorHandlingPolicy>	
             </ErrorHandlingPolicies>
          </ErrorHandlingConfiguration>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-n: Optional element that can occur one or more times.

Element relationships

Relationship Element
Parent elements ErrorCodes
Child elements None

Remarks

For a list of predefined error codes, see Predefined error codes for the data pipeline.

Declare the ErrorCodes element only within a service manifest of the , , , and applications.
