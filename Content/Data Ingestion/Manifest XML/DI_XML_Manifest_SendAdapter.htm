SendAdapter element

 

Declares the adapter used to send messages to the database.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <Feeds>
             <Feed Name="String">
                <SendAdapter>...</SendAdapter>
             </Feed>	
          </Feeds>
       ]]>
    </DefaultValue> 
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Feed
Child elements None

Attributes

Attribute Description
Name

The following value is allowed.

Value Description
Microsoft.Amalga.Pipeline.AcquisitionAdapters.SqlStorage Defines the MSG201 table for an instance of the database.

Remarks

The SendAdapter element is mandatory in the service manifest of the . Do not declare this element in the manifests of other pipeline applications.
