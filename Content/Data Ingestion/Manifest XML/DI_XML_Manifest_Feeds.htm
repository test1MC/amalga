Feeds element

 

Declares a collectiInteracting with Data Change Notificationon element in which you define one source system feed.

SyntaxElements element

[ <Setting Name="FeedConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <Feeds>
             <Feed />
          </Feeds>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements DefaultValue
Child elements Feed

Remarks

The Feeds element is mandatory as the root element of the FeedConfigurationXML setting for all service manifests. Do not declare this element in the manifests of other pipeline applications.

A single child Feed element declaration is mandatory.
