IdleSleepSeconds setting

 

Sets the number of seconds to pause when there is no data to be processed.

Syntax

[ <Setting Name="IdleSleepSeconds" Type="System.Int32">
   <DefaultValue>...</DefaultValue>
</Setting> ]

Setting characteristics

Characteristic Description
Data type and maximum value Integer with a maximum value of 600 (10 minutes)
Default value None
Target applications

Data Parser (service manifest), Message Shredder (service manifest)

Example

The following example sets the idle sleep time to 5 seconds. If there is no activity during the period, the associated process will sleep until the onset of new activity.

<Setting Name="IdleSleepSeconds" Type="System.Int32">
                <DefaultValue>5</DefaultValue>
</Setting>
