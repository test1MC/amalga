ErrorTitle element

 

Declares an exception title to be used in the log entry.

SyntaxExceptionType element

[ <Setting Name="ErrorHandlingConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <ErrorHandlingConfiguration>
             <ErrorHandlingPolicies>
                <ErrorHandlingPolicy>
                   <LogSetting>
                      <ErrorTitle>...</ErrorTitle>
                   </LogSetting>
                </ErrorHandlingPolicy>	
             </ErrorHandlingPolicies>
          </ErrorHandlingConfiguration>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements LogSetting
Child elements None

Remarks

The ErrorTitle element is mandatory to properly display the error in the event log. Be sure that the title is meaningful to system administrators.

Declare the ErrorTitle element only within a service manifest of the , , , and applications.

Example

<ErrorTitle>DbServerNotResponding</ErrorTitle>
