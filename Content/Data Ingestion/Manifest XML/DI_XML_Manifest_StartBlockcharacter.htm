StartBlockCharacter element

 

Declares the start block character for the HL7 messages in a feed.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <Feeds>
             <Feed> Name="String"
                <ReceiveAdapter> Name="String"
                   <StartBlockCharacter>...</StartBlockCharacter>
                </ReceiveAdapter>
             </Feed>	
          </Feeds> 
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements ReceiveAdapter
Child elements None

Remarks

Constraints

Use the StartBlockCharacter element to configure an instance of the when the message transport format is MLLP. Before applying this element tag, set the Name attribute value of the ReceiveAdapter element to "Microsoft.Amalga.Pipeline.AcquisitionAdapters.MLLPServer".

Example

<startblockcharacter>0x0B</startblockcharacter>
