ErrorHandlingPolicy element

 

Declares an error-handling policy for the specified exception.

Syntax

[ <Setting Name="ErrorHandlingConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <ErrorHandlingConfiguration>
             <ErrorHandlingPolicies>
                <ErrorHandlingPolicy>
                   <ExceptionType>...</ExceptionType>
                   <ErrorCodes/ >
                   <Action>...<Action>
                   <RetryInterval>...</RetryInterval>
                   <MaximumRetry>...</MaximumRetry>
                   <FinalAction>...<FinalAction>
                   <LogSetting>...<LogSetting>
                </ErrorHandlingPolicy>	
             </ErrorHandlingPolicies>
          </ErrorHandlingConfiguration>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-n: Optional element that can occur one or more times.

Element relationships

Relationship Element
Parent elements ErrorHandlingPolicies
Child elements Action, ErrorCodes, ExceptionType, FinalAction, LogSetting, MaximumRetry, RetryInterval

Remarks

The ErrorHandlingPolicy element can overwrite the actions for some predefined errors and can also define a new policy for a private exception.

Declare the ErrorHandlingPolicy element only within a service manifest of the , , , and applications.

Example

The following example defines the policy for the System.Data.SqlClient.SqlException exception and defines the format of errors in the log.

<ErrorHandlingConfiguration>
  <ErrorHandlingPolicies>    
    <ErrorHandlingPolicy>
      <ExceptionType>System.Data.SqlClient.SqlException</ExceptionType>
      <ErrorCodes>
        <ErrorCode>7</ErrorCode>
        <ErrorCode>9</ErrorCode>
      </ErrorCodes>     
      <Action>Stop</Action>
      <LogSetting>
        <EventId>33001</EventId>
        <ErrorTitle>DbServerNotResponding</ErrorTitle>
        <ErrorDescription>The component has attempted to connect to the Database Server; however, the connection has timed out.</ErrorDescription>
      </LogSetting>
    </ErrorHandlingPolicy>
  </ErrorHandlingPolicies>    
</ErrorHandlingConfiguration>
