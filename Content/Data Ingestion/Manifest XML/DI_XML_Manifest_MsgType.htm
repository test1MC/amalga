MsgType element

 

Declares a value that represents the MsgType column in the MSG201 table of the database for the feed.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <Feeds>
             <Feed>
               <MsgType>...</MsgType>
             </Feed>	
          </Feeds>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Feed
Child elements None

Remarks

The MsgType element is mandatory in all feed service manifests. Do not declare this element in the manifests of other pipeline applications.

The MsgType element value populates the MsgType column in the database MSG201 table. The message type must match across all pipeline applications. Use the Type element in the and applications to match the message type.

Example

<msgtype>LAB</msgtype>
