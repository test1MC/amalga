IdleConnectionTimeout setting

 

Sets the number of seconds for shredding to pause when there is no data to be processed.

Syntax

[ <Setting Name="IdleConnectionTimeout" Type="System.Int32">
   <DefaultValue>...</DefaultValue>
</Setting> ]

Setting characteristics

Characteristic Description
Data type and range Integer with a valid range of 5000 to 10000 milliseconds.
Default value None
Target applications

Feed Acquisition (service manifest)

Example

A value of 5000 milliseconds (5 seconds) should match the connection requirements of the source system and can be used as a default value in most cases. In the following example, a new connection will replace the old connection in 5 seconds if the old connection remains idle during the waiting time.

<Setting Name="IdleConnectionTimeout" Type="System.Int32">
                <DefaultValue>5000</DefaultValue>
</Setting>
