SourceType element

 

Declares whether the plug-in assembly comes from a file system or source.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <DataParserConfiguration>
            <Plugins>
               <Plugin>
                  <SourceType>...</SourceType>				
               <Plugin>			
            <Plugins>
         </DataParserConfiguration>
      ]]>
   </DefaultValue>
</Setting> ]
[ <Setting Name="PluginModelConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <Plugin>
            <SourceType>...</SourceType>				
         <Plugin>		
      ]]>
   </DefaultValue>
</Setting> 
[ <Setting Name="ShredderPluginConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <Plugin>
            <SourceType>...</SourceType>				
         <Plugin>		
      ]]>
   </DefaultValue>
</Setting> ]

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-1: Optional element that can occur once and only once

Element relationships

Relationship Element
Parent elements Plugin
Child elements None

Remarks

Valid values for the SourceType element are FileSystem or Fabric. For information about extending pipeline applications with plug-in code, including examples, see Extend Pipeline applications.
