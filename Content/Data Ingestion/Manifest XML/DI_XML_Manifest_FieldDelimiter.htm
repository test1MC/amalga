FieldDelimiter element

 

Declares the field delimiter for a comma-separated values (CSV) shredder.

Syntax

[ <Setting Name="CustomizedConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <CSVConfiguration>
            <FieldDelimiter />
         </CSVConfiguration>
      ]]>
   </DefaultValue>
</Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value Comma (,)
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements CSVConfiguration
Child elements None

Remarks

Declare the FieldDelimiter element in a CSV-shredder service manifest to specify an alternative delimiter. Do not declare this element in the manifests of other pipeline applications.

Example

<FieldDelimiter>~</FieldDelimiter>
