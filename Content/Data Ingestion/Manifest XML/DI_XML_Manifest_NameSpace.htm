NameSpace element

 

Declares the namespace of the plug-in code.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <DataParserConfiguration>
            <Plugins>
               <Plugin>
                  <NameSpace>...</NameSpace>				
               <Plugin>			
            <Plugins>
         </DataParserConfiguration>
      ]]>
   </DefaultValue>
</Setting> ]
[ <Setting Name="PluginModelConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <Plugin>
            <NameSpace>...</NameSpace>				
         <Plugin>		
      ]]>
   </DefaultValue>
</Setting> 
[ <Setting Name="ShredderPluginConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <Plugin>
            <NameSpace>...</NameSpace>				
         <Plugin>		
      ]]>
   </DefaultValue>
</Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once

Element relationships

Relationship Element
Parent elements Plugin
Child elements None

Remarks

For information about extending pipeline applications with plug-in code, including examples, see Extend Pipeline applications.
