Name element

 

Declares the name of a particular feed.

Syntax

[ <Setting Name="FeedInfoXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <FeedInfo> 
             <Name>...</Name>
          </FeedInfo>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements FeedInfo
Child elements None

Remarks

A feed is an individual source of data that you can distinguish from other feeds with a name. The feed name must be unique and should conform to a standardized naming convention.

Always declare the Name element in the service manifest of the and applications. Do not declare this element in the manifests of other pipeline applications.

Example

<Name>ADT Feed</Name>
