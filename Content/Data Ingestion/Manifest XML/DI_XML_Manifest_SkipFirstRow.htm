SkipFirstRow element

 

Declares whether the first row of a comma-separated values (CSV) message can be skipped.

Syntax

[ <Setting Name="CustomizedConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <CSVConfiguration>
            <SkipFirstRow />
         </CSVConfiguration>
      ]]>
   </DefaultValue>
</Setting> ]

Element characteristics

Characteristic Description
Data type and length Boolean
Default value False
Cardinality 0-1: Optional element that can occur once and only once

Element relationships

Relationship Element
Parent elements CSVConfiguration
Child elements None

Remarks

Declare the SkipFirstRow element in a CSV-shredder service manifest when you want avoid shredding the first row, which might contain the CSV header. Otherwise, the first row will be shredded. Do not declare this element in the manifests of other pipeline applications.

Example

<SkipFirstRow>true</SkipFirstRow>
