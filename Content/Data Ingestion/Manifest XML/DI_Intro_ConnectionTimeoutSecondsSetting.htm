ConnectionTimeoutSeconds setting

 

Sets the number of seconds to wait before the connection times out.

Syntax

[ <Setting Name="ConnectionTimeoutSeconds" Type="System.Int32">
   <DefaultValue>...</DefaultValue>
</Setting> ]

Setting characteristics

Characteristic Description
Data type and maximum value Integer with a maximum value of 600 (10 minutes).
Default value None
Target applications

Feed Acquisition (service manifest), Data Parser (service manifest)

Example

The following example sets the connection timeout to one minute (60 seconds).

<Setting Name="ConnectionTimeoutSeconds" Type="System.Int32">
                <DefaultValue>60</DefaultValue>
</Setting>
