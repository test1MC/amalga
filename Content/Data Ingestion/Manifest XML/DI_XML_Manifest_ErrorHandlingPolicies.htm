ErrorHandlingPolicies element

 

Declares one or more error-handling policies.

Syntax

[ <Setting Name="ErrorHandlingConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <ErrorHandlingConfiguration>
             <ErrorHandlingPolicies>
                <ErrorHandlingPolicy />	
             </ErrorHandlingPolicies>
          </ErrorHandlingConfiguration>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements ErrorHandlingConfiguration
Child elements ErrorHandlingPolicy

Remarks

Declare the ErrorHandlingPolicies element only within a service manifest of the , , , and applications.
