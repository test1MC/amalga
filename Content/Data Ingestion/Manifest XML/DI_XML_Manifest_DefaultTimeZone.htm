DefaultTimeZone element

 

Declares the time zone appropriate for the shredded data.

Syntax

[ <Setting Name="FeedInfoXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <FeedInfo> 
             <DefaultTimeZone>...</DefaultTimeZone>
          </FeedInfo>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements FeedInfo
Child elements None

Remarks

Select a time zone that is meaningful at your site or for the feed. Declare the DefaultTimeZone element in the service manifest of the and applications. Do not declare this element in the manifests of other pipeline applications.

Example

<DefaultTimeZone>Pacific Standard Time</DefaultTimeZone>
