Terminology lookup functions

 

The following table describes the functions that perform terminology lookup operations when declared in service manifests of the Data Parser application.

Function name Description

Lookups.usf_Terminology_ConvertCodeToCode

Maps an incoming terminology code to a destination terminology code.
Lookups.usf_Terminology_ConvertCodeToText Returns text (terminology meaning) for the specified terminology code.
Lookups.usf_Terminology_ConvertTextToCode Returns the first matching HLI concept GUID for the specified text (terminology meaning).
