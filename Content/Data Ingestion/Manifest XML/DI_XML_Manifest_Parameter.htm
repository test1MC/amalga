Parameter element

 

Declares a parameter for the specified action.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         </DataParserConfiguration>
					
            <InputParameters>
               <Parameter Sourcetype="String">
                  <Array />
               </Parameter>
            </InputParameters>

         </DataParserConfiguration>
      ]]>
   </DefaultValue>
</Setting> ]

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-n: Optional element that can occur one or more times

Element relationships

Relationship Element
Parent elements InputParameters
Child elements Array

Attributes

Attribute Description
Name

(Optional string.) Name of the parameter.

SourceType

(Required string.) Defines the input source. The following values are allowed.

Value Description
ShreddedInput

Reads input from a shredded message. You must also specify values for the SourceTable and SourceName attributes in the parameter.

Always explicitly declare SourceTable and SourceName attribute values in the parser service manifest using the MessageShredderInput element before adding them to a parameter.
DerivedField Reads a field that you derive in the parser service manifest. You must also specify values for the SourceTable and SourceName attributes in the parameter.
Inline Reads a value that you specify in the parameter using the Value attribute.
Constant Reads a global value. You must declare the constant first in the DataConstants element and then specify the named constant with the SourceName attribute.
Array Reads a variable list of arguments that you declare with the Array element. Use the Type attribute to indicate the array type.
SourceTable (Optional string) Specifies the table from which to read the data. For example, if you want to define a transform for table PID, set the SourceTable attribute to "PID" or any of its parents (MSH).
SourceName (Optional string) The name of the table column or constant as indicated by the source type.
Type (Optional string) Data type such as System.String.
Value (Optional string) Specifies an explicit inline value, such as "|" (pipe).

Remarks

The Parameter element provides an individual parameter as input to a transform action, a lookup function, EnterpriseId action, or a condition action. The number and order of parameters depends on the requirements of the specified transform, function, or condition.

Declare the Parameter element only within a service manifest of the . Do not declare this element in the manifests of other pipeline applications.

Example

<Element Name="Diagnosis_DG1_ContextKey" Type="System.String" MaxLength="200" ContextKey="true">
    <ConceptId>f/xuQgEy71iABYBuCrJmYw</ConceptId>
    <Transform>
        <Action>Concatenate</Action>
        <InputParameters>
            <Parameter Name="Pipe" SourceType="Constant" SourceName="Pipe" />
            <Parameter Name="Visit" SourceType="ShreddedInput" SourceTable="PV1" SourceName="PV1_19" />
            <Parameter Name="SetId" SourceType="DerivedField" SourceTable="DG1" SourceName="SetId" />
        </InputParameters>
    </Transform>
</Element>
