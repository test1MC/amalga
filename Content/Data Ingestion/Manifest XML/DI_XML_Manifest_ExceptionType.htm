ExceptionType element

 

Declares a predefined or private exception type.

Syntax

[ <Setting Name="ErrorHandlingConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <ErrorHandlingConfiguration>
             <ErrorHandlingPolicies>
                <ErrorHandlingPolicy>
                   <ExceptionType>...</ExceptionType>
                </ErrorHandlingPolicy>	
             </ErrorHandlingPolicies>
          </ErrorHandlingConfiguration>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements ErrorHandlingPolicy
Child elements None

Remarks

The ExceptionType element is mandatory to identify the type of exception. All predefined exceptions are of type System.Data.SqlClient.SqlException.

Declare the ExceptionType element only within a service manifest of the , , , and applications.

Example

<ExceptionType>System.Data.SqlClient.SqlException</ExceptionType>
