CustomizedConfigurationXML setting

 

The following hierarchy describes the XML elements needed to customize a comma-separated values (CSV) application configuration.

<Setting Name="CustomizedConfigurationXml" Type="System.String">
   <DefaultValue> [CDATA]
      <CSVConfiguration>
         <RowDelimiter>
         <FieldDelimiter>
         <Quote>
         <SkipFirstRow>
