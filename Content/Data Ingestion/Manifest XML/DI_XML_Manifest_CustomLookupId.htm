CustomLookupId element

 

Declares the custom lookup value of a source identifier parameter.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <DataParserConfiguration>
             <EnterpriseId>
               <EnterpriseIdAction TransformName="String" ArchorTable="String">
                  <InputIdentifiers>
                     <EncounterIdentifier Name="String">
                        <CustomLookupId>...</CustomLookupId>
                     </EncounterIdentifier>
                     <EncounterSetIdentifier Name="String">
                        <CustomLookupId>...</CustomLookupId>
                     </EncounterSetIdentifier>
                     <PatientRecordIdentifier Name="String">
                        <CustomLookupId>...</CustomLookupId>
                     </PatientRecordIdentifier>
                     <PersonIdentifier Name="String">
                        <CustomLookupId>...</CustomLookupId>
                     </PersonIdentifier>
                     <ProviderIdentifier Name="String">
                        <CustomLookupId>...</CustomLookupId>
                     </ProviderIdentifier>
                  </InputIdentifiers>
               </EnterpriseIdAction>	
            </EnterpriseId>
          </DataParserConfiguration>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length ParameterConfiguration (String?)
