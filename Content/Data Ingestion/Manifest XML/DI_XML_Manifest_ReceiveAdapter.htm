ReceiveAdapter element

 

Declares an adapter that identifies the port or URL used by a instance to acquire data and to acknowledge the delivery to the source system.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
    <DefaultValue>
       <![CDATA[<?xml version="1.0" encoding="utf-8"?>
          <Feeds>
             <Feed Name="String">
                <ReceiveAdapter Name="String">
                   <Port>...</Port>
                   <StartBlockCharacter>...</StartBlockCharacter>
                   <EndBlockCharacter>...</EndBlockCharacter> 
                   <CarriageReturnCharacter>...</CarriageReturnCharacter>
                   <AckCapture>...</AckCapture> 
                   <AckFormular>...</AckFormular> 
                   <NackCapture>...</NackCapture> 
                   <NackFormular>...</NackFormular> 
                   <DefaultAck>...</DefaultAck> 
                   <DefaultNack>...</DefaultNack> 
                   <Directory>...</Directory> 
                   <SearchPattern>...</SearchPattern>
                </ReceiveAdapter>
             </Feed>	
          </Feeds>
       ]]>
    </DefaultValue>
 </Setting> ]

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 1-1: Required element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Feed
Child elements AckCapture, AckFormular, CarriageReturnCharacter, DefaultAck, DefaultNack, Directory, EndBlockCharacter, NackCapture, NackFormular, Port, SearchPattern, StartBlockCharacter

Attributes

Attribute Description
Name

The following values are allowed.

Value Description
Microsoft.Health.Acquisition.Adapters.MLLPServer

Minimal Lower Layer Protocol (MLLP) to transport HL7 messages.
Microsoft.Health.Acquisition.Adapters.FileAdapter

File format to transport file-based messages, such as comma-separated values (CSV) format.

Remarks

The ReceiveAdapter element is mandatory in the service manifest of the . Do not declare this element in the manifests of other pipeline applications.

Required and optional child elements are scoped by the adapter name as follows:

  • Microsoft.Amalga.Pipeline.AcquisitionAdapters.MLLPServer

    • Port (required)
    • StartBlockCharacter
    • EndBlockCharacter
    • CarriageReturnCharacter
    • AckCapture
    • AckFormular
    • NackCapture
    • NackFormular
    • DefaultAck
    • DefaultNack

  • Microsoft.Amalga.Pipeline.AcquisitionAdapters.FileAdapter

    • Directory (required)
    • SearchPattern (required)
