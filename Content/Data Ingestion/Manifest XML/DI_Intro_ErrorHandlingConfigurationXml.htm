ErrorHandlingConfigurationXML setting

 

The following hierarchy describes the XML elements available to configure error handling for pipeline and materialization applications.

<Setting Name="ErrorHandlingConfigurationXml" Type="System.String">
   <DefaultValue> [CDATA]
      <ErrorHandlingConfiguration> 
         <ErrorHandlingPolicies>
            <ErrorHandlingPolicy>
               <ExceptionType> 
               <ErrorCodes> 
                  <ErrorCode> 
               <Action> 
               <RetryInterval>
               <MaximumRetry>
               <FinalAction>
               <LogSetting> 
                  <EventId> 
                  <ErrorTitle>
                  <ErrorDescription>
