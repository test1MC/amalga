Dependency element

 

Declares a plug-in application dependency on another assembly.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <DataParserConfiguration>
            <Plugins>
               <Plugin>
                  <Dependencies>
                     <Dependency>...<Dependency>
                  </Dependencies>		
               </Plugin>			
            </Plugins>
         </DataParserConfiguration>
      ]]>
   </DefaultValue>
</Setting> ]
[ <Setting Name="PluginModelConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <Plugin>
            <Dependencies>
               <Dependency>...</Dependency>
            </Dependencies>					
         </Plugin>		
      ]]>
   </DefaultValue>
</Setting> 
[ <Setting Name="ShredderConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <Plugin>
            <Dependencies>
               <Dependency>...</Dependency>
            </Dependencies>					
         </Plugin>		
      ]]>
   </DefaultValue>
</Setting>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-n: Optional element that can occur one or more times.

Element relationships

Relationship Element
Parent elements Dependencies
Child elements None

Attributes

Attribute Description
Assembly

(Required string.) Name of the assembly on which the plug-in depends.
URL (Required string) Location of the required assembly.

Remarks

For information about extending pipeline applications with plug-in code, including examples, see Extend Pipeline applications.
