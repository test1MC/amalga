BatchSize setting

 

Sets the number of rows processed during each shredder or parser cycle.

Syntax

[ <Setting Name="BatchSize" Type="System.Int32">
   <DefaultValue>...</DefaultValue>
</Setting> ]

Setting characteristics

Characteristic Description
Data type and maximum value Integer with a maximum value of 10000 rows.
Default value 500 rows
Target applications

Data Parser (service manifest), Message Shredder (service manifest)

Example

The following example sets the batch size to 50 rows.

<Setting Name="BatchSize" Type="System.Int32">
                <DefaultValue>50</DefaultValue>
</Setting>
