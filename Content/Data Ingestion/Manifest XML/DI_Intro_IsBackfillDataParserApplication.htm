IsBackfillDataParserApplication setting

 

Sets whether the instance of a will parse historic data.

Syntax

[ <Setting Name="IsBackfillDataParserApplication" Type="System.Int32">
   <DefaultValue>...</DefaultValue>
</Setting> ]

Setting characteristics

Characteristic Description
Data type and range Boolean
Default value False
Target applications

Backfill Data-Parser (application manifest)

Example

The following example enables backfill parsing. For complete instructions, see Enable backfill parsing.

   <Setting Name="IsBackfillDataParserApplication" AmalgaDisplayName="DataParser Application Setting" Type="System.Boolean" >
         <DefaultValue>true</DefaultValue>   
   </Setting>
