FeedInfoXML setting

 

The following hierarchy describes the XML elements used to configure and application instances.

The Type element value must match the MsgType element value of the acquired feed.

<Setting Name="FeedInfoXml" Type="System.String"> 
   <DefaultValue> [CDATA]
      <FeedInfo>
         <Name>
         <Type>
         <Provider>
         <TerminologyName>
         <PreferenceRanking>
         <PreferredCulture>
         <DefaultTimeZone>
