ResultTypeSortField element

 

Declares a string with which to sort the resulting rows of a collapsed view.

Syntax

[ <Setting Name="FeedConfigurationXml" Type="System.String">
   <DefaultValue>
      <![CDATA[<?xml version="1.0" encoding="utf-8"?>
         <DataParserConfiguration>
            <DataParserOutput>
               <Segment>
                  <CollapseConfiguration>
                     <ResultTypeSortField>...</ResultTypeSortField>
                  </CollapseConfiguration>
               </Segment>
            </DataParserOutput>
					
            <ExtensibleDataParserOutput>
               <Segment>
                  <CollapseConfiguration>
                     <ResultTypeSortField>...</ResultTypeSortField>
                  </CollapseConfiguration>
               </Segment>
            </ExtensibleDataParserOutput>
					
            <ExpandableDataParserOutput>
               <Segment>
                  <CollapseConfiguration>
                     <ResultTypeSortField>...</ResultTypeSortField>
                 </CollapseConfiguration>
              </Segment>
            </ExpandableDataParserOutput>
         </DataParserConfiguration>
      ]]>
   </DefaultValue>
</Setting> ]

Element characteristics

Characteristic Description
Data type and length String
Default value Sorts by the RowId column.
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements CollapseConfiguration
Child elements None

Remarks

The ResultTypeSortField element provides an optional string value to apply to the sorting algorithm. Declare the value optionally in the data-parser service manifest. Do not declare this element in the manifests of other pipeline applications.

Example

The following example specifies collapse-view results to sort and filter.

<CollapseConfiguration>
      <PrimaryKeyColumn>Order_OBR_ContextKey</PrimaryKeyColumn>
      <ResultTypeSortField>CAST(REPLACE(REPLACE(REPLACE([OBR_25], 'C', 100), 'F', 50), 'P', 1) as int)</ResultTypeSortField>
      <ResultRowsFilter>[OBR_25] IN ('C', 'F', 'P')</ResultRowsFilter>
 </CollapseConfiguration>
