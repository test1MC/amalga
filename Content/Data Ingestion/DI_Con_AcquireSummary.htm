Remarks

The moves feed data from a source interface to the .dbo.MSG201 table. Each service instance consists of one configurable receive adapter and an independent send adapter that identifies the destination message queue. You can configure the receive adapter to accept one of the following message-delivery formats:

  • File format to transport file-based messages, such as rows in a comma-separated values (CSV) file. You must specify a feed name, feed type, and the source directory (with a search pattern) of the file.
  • Minimal Lower Layer Protocol (MLLP) to transport HL7 messages. You must specify a feed name, feed type, and unique port number. The receive adapter returns acknowledgement ACK and NACK messages for the HL7 feed.

Feed transport into the pipeline can take place over the Internet, a wide-area network (WAN), or a local-area network (LAN). All modes deliver high reliability, high throughput, and high availability.
