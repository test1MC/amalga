Shredder input

During every polling interval, a Message Shredder application instance reads a batch of raw messages from MessageQueue.dbo.MSG201. You can specify the number of messages in the batch for each message type. The default batch size is 500.

Format example: Comma-separated value (CSV)

item1, item2, item3, item4

item5, item6, item7, itme8
