Acquisition output

 

Acquired data appears in the MSG201 database table of the message queue specified in the acquisition service manifest. Valid output is ready for message shredding.

Failed messages

Failed messages are routed by message type to the MessageQueue.dbo.MSG201 table or secondary storage with an .ERR file extension. For example, if the normal message type is ADT, the failed message is stored as ADT.ERR.
