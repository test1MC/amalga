Manage the module

 

This topic describes how you can control the display of the module.

The module displays temperature, pulse, blood pressure, SpO2, respiratory rate, and weight over the selected time range.

Ref# Name Description and tasks
1 TIME RANGE Allows you to choose the time range displayed in the graph.
2 Time interval label Allows you to vary the length of time that is shown in the graph.
3 Date slider Allows you to move forward or backward within the time frame.
