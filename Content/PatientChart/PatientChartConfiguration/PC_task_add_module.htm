Add a module

 

This topic describes how to add a module to using the Patient Chart Configuration.

Using the Patient Chart Configuration module, you can add and delete modules, add and delete widgets, and change the configuration for modules and widgets.

To add a module

  1. Click .

  2. Click Add.

  3.  Complete the form using the information in the table below.
  4. Click Save.

Configuration information

Element Mandatory? Comments
ModuleID Yes

System-generated unique ID; can be modified in the XML file, but must be unique; should not contain any punctuation included in filtering set (see Punctuation filtering)
DisplayName Yes A unique name is recommended; should not contain any punctuation included in filtering set; Resource key is supported (see Resource file)
LaunchURL Yes ${Prefix} is required
IsRecommended No Default = false; If true, module will be displayed in the Recommended group in the module launcher
HasChildren No Must be true if the module has sub-modules

Punctuation filtering

The following punctuation will be replaced with an empty string:

  • < > \ ^ " ' [ ] { } or a space

Do not include them in IDs or names.

Resource file

The resource file prefix is Microsoft.Health.PatientChart.Configuration. It is located in the application global resource folder - for example, C:\inetpub\wwwroot\wss\VirtualDirectories\8080\App_GlobalResources.
