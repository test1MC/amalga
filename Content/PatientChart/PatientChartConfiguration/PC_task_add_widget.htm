Add a widget

 

This topic describes how to add a widget to using the Patient Chart Configuration.

Using the Patient Chart Configuration, you can add and delete modules, add and delete widgets, and change the configuration for modules and widgets.

To add a widget

  1. Click Clinical Summary.
  2. Click Add.

  1. Complete the form using the information in the table below.

  2. Click Save.
  3. Click the name of the widget.
  4. Click the PresentationConfig tab.
  5. Complete the configuration using the information in the table below.
  6. Click Save.
  7. Click the LogicalConfiguration tab.
  8. Complete the configuration using the information in the table below.
  9. Click Save.
