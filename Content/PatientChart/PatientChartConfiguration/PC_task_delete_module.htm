Delete a module

 

This topic describes how to delete a module from .

Using the Configuration, you can remove a module from the list of available modules for the .

To delete a module

  1. Open the Configuration.
  2. Expand .
  3. Click the name of the module to be deleted.
  4. Click Delete.

  5. Click OK to confirm the deletion.

    You can delete all but the last module using the Patient Chart Configuration. To delete the last module, modify the Patient Chart configuration file.

    When you delete a module, the configuration will also be deleted. It is recommended that you backup your configuration prior to deleting a module.
