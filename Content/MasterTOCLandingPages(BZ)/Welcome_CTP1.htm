Introduction to documentation

 

This topic introduces you to the key feature areas of this release.

See New and updated content for a list of topics that have changed since the initial release of the community technical preview.

Welcome to . This community technical preview showcases a development and deployment experience that embraces core Microsoft tools and technologies for data-driven and service-oriented architectures. It enables you to work from a set of data entities that you expose in virtual schemas and bind to data assets at your site. The development environment provides a rich set of templates and tools for developing services, sites, and user interfaces for applications. With these tools, you build user interfaces for your site on the platform, adding controls from the to pages built from templates.

This release also introduces approaches to system and application management designed to create a consistent experience for system administrators. The security model enables granular role-based control at the level of the system, application, and data entity.
