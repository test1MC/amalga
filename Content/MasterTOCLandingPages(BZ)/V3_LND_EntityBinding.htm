Entity binding configuration

 

This section includes instructions for defining entity bindings using . For information about defining entity bindings using , see Set-AmEntityBinding and Entity binding XML reference.

Configuring entity binding for a binding source involves first binding the primary key property of the entity to a primary key column of a table or view. When the primary key property of the entity is bound, you then bind other entity properties to either selected columns or property-specific binding options, such as constants or NULL values.
