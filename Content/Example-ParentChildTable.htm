Lowest Level Elements

These elements do not have child elements.

Format

<Element Name>    

Optional [or Required]. Description.
Parent element: <Parent>

Examples

<CellMinHeight>    
Optional. Sets the minimum height of cells.
Parent element:  <UIParameters>
<CellMinWidth>    
Optional. Sets the minimum width of cells.
Parent element:  <UIParameters>

Elements with Parent and Child Elements

Example

<RowTemplates>    
Required. Contains the elements that configure the organization and appearance of row headers for each panel in the <TrendPanel> or <DetailPanel> tabs.
Parent element: <UIParameters>
Child elements: <HeaderCellTemplate>, <Level>, <Snippet>

Elements with Parent and Child Elements and Attributes

Example

<HeaderCellTemplate>    
Required. Contains configuration information for header cells.
Parent element: <RowTemplates>
Child elements: <Level>, <Snippet>, <HoverItems>
Attribute Description Values
