Inserting a Forward Link into a Topic

To insert a forward link into a topic

  1. Check the Forward link list in the \Contents folder in your Flare project.
  2. If you don’t find the link you need, request one from Laura Lee (v-labenn).
  3. Create a hyperlink with the go link as the Website and appropriate text as the Link Text.
  4. After the hyperlink, type: [at symbol]go link.
  5. Add a hyperlink to the go link text, with the go link in both the Link Text and the Website boxes. Condition this PrintOnly.

9. Check the Forward link list in Contents in your Flare project.

10. If you don’t find the link you need, request one from Cindy.

11. Create a hyperlink with the go link as the Website and appropriate text as the Link Text.

12. After the hyperlink, type: at go link.

13. Add a hyperlink to the go link text, with the go link in both the Link Text and the Website boxes. Condition this PrintOnly.

The print only text is for a printed PDF. Be sure to do a print preview with and without the PrintOnly condition excluded, to verify that the necessary and sufficient spaces are in place.

Example

In the following example, the parts are styled and linked as follows:

• “Microsoft Download Center” is the hyperlink that will show for WebHelp.

• “ at http://go.microsoft.com/fwlink/?LinkId=157756” is conditioned print only (note the initial space).

• “http://go.microsoft.com/fwlink/?LinkId=157756” is a hyperlink with the hyperlink text the same as the go link, so that if a PDF page is printed out, the reader will be able to type in the go link and access the correct site.

Microsoft .NET Framework Version 3.5 SP1 is available at the Microsoft Download Center.
