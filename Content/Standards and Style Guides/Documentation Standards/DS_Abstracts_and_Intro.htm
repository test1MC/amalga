Introduction

Abstract

This document defines practices and standards for authoring content to support the release of Amalga v3. All writers working in the Microsoft Health Solutions Group for the Amalga v3 platform and solutions teams are expected to adhere to the requirements set forth in this plan.

Overview

This plan defines the requirements for creating and publishing clinical user, IT professional, and developer documentation. It defines tools and processes that pertain to authors. The following list defines the information covered by this plan:

  • Style and usage resources
  • Tools and process used by writers
  • Project organization in source repository
  • Deliverable types and formats
  • File naming convention
  • Variable organization and usage
  • Tag sets
    • Editing Workflow
    • Topic Status
  • Linking strategy
  • Table of Contents organization
  • Topic types
  • Indexing
  • Art
