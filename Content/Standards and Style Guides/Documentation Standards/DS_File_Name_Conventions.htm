File name conventions

File names in the source repository for topic files, image files, snippets, and other resources should include human-readable abbreviations for the following:

  • Feature name
  • Topic type
  • Topic title

For example, a task topic that tells readers how to bind an entity to the virtual schema might be:

VSchema_Task_BindEntity.xml

Topic type prefixes 

If that task topic is created with the Procedures (PROC) topic template, add a prefix to the file name to indicate the topic type: PROC_VSchema_Task_BindEntity.xml

 

See Topic Types for a list of abbreviations for each topic type.
