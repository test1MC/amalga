TopicTypes

For the latest on topic types, see the file tag set for TopicType; for topic templates, see C:\UA-SRC\StyleMaster-v3\Content\Standards and Style Guides\Templates\Amalga Templates\Content\.

The detailed specification of the structure of each topic type is under development. When they are complete, you will see them as a topic templates option when you create a new file:  use one of the templates when creating documentation.

Topic types are still in the process of being defined. Types come from either Laura Lee's list (LLB), the file tag set (FTS), Documentation Standards (DS), or all three (3).
