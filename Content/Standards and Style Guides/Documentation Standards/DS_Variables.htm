The Amalga UA team uses the variables feature in MadCap Flare to manage volatile terminology such as feature names, user interface labels, file paths, and other often-used items subject to change during product development. Variables are also used to make sure the right legal text is used, for instance, for brand or trademarked names.

Process

Writers specify the variables they need for their features on an ongoing basis and communicate these needs to the editors, supplying the project name, variable text, and a short description of what the variable is.

The editors review the suggestions and create the variables in Flare in the Stylemaster or StyleMaster-V3 project, and imports the changes to the appropriate projects.l

Variable Sets

The following variable sets are available in Flare for the Amalga V3 documentation effort:

  • Feature Name

    Set name: FeatureName

    Usage: To be defined

  • File Names

    Set name: Filenames

    Usage: Commonly used files types, such as Amalga Pack (.apk) and file names such as Doors Installer package (.msi)

  • HSG Branding

    Set name: BrandingHSG

    Usage: Brand names associated with the HSG product line, such as Amalga or HealthVault; includes TM bugged variables

  • Imaging

    Set name: Imaging

    Usage: For terms related to

  • Legal

    Set name: LegalAnd3rdParty

    Usage: Primarily in legal notices

  • Manual Names

    Set name: ManualNames

    Usage: Collection of files with related subject matter -- typically a major TOC section (in Web Help) or a PDF output produced by a single target

  • Microsoft Branding

    Set name: BrandingMS

    Usage: Brand names of Microsoft products used in the HSG product line, such as Doors Server or .

  • Paths

    Set name: Paths

    Usage: Complex paths such as Main Registry Key (HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Amalga\2.0) or long folder names such as the Product name for installation media path (MicrosoftAmalgaUnifiedIntelligenceSystem2009)

  • Technical Terms

    Set name: TechnicalTerms

    Usage: To be defined.

  • UI Buttons

    Set name: UI-Buttons

    Usage: Text on buttons

  • UI Labels

    Set name: UI-Labels

    Usage: all UI text not on buttons
