Create topics with Amalga templates

 

This topic describes how to create a topic with an Amalga template.

The topic templates for AmalgaV3 can be found in Content Explorer, under \Standards and Style Guides\Templates\Amalga Templates. Make sure you specify the root directory where you want the topic to appear in your project.

To create a topic with an Amalga Template

  1. Select Project > Add Topic, select the template from those listed in New from template:, and then specify the file name. Be sure to follow standard naming conventions with a standard prefix (such as "CON_" for Conceptual topic type, "PROC_" for Procedure topic type, and so on).

2.In the Add New Topic dialog box, click the down arrow, and then re-select AmalgaMain as the style (.

3

  1. 2. [EXTRA STEP] In the Add New Topic dialog box, click the down arrow, and then re-select AmalgaMain as the style sheet.

Since the Amalga V3 project includes building the API docs, which use the Presentation.css stylesheet, you must always explicitly declare the AmalgaMain.css stylesheet. If you follow this extra step, then the CSS link is adjusted according to the topic’s location relative to the CSS.
