Add a work item in Visual Studio Team Foundation Server (TFS)

This topic describes how to add a work item (create a bug) in TFS.

File a bug in TFS

If you are not already connected to your project, you will need to start TFS.

To connect to your project in Visual Studio Team Foundation Server (TFS)

  1. Start , and then click Connect to Team Foundation Server.
  2. Click Connect.

To add a work item (bug) in TFS

  1. In the SH-BJ project, select Work Items, and then right-click and select New Work Item > Bug. The following dialog box is displayed:

  2. Fill in (or select a value for) the required (highlighted) fields, as listed below:
Field Value
Title Example: “Build Review Doc Fixes”
Assigned To <writer_alias> Example: "v-labenn"
Issue Content Defect
Priority 3
Severity 3
How found Manual Testing
Release Note No
Details Summarize what the bug is about —the issue or problem, and what action is needed to resolve it, if known. If detail is in an attached file, add the phrase“See attached file”.
  1. 3. In the Details tab, enter the pertinent information for the bug.
    1. a) If there is a screen shot or other information, such as an email thread, illustrating the doc bug, click the Attachments tab, and then click the Add button.
    2. b) Enter the path for (or copy and paste) the file (such as document, screen shot or email thread).

4. From the File menu, click Save New Bug <xxx>.

5. Optionally, email the writer with any information that was not appropriate to put in the bug.

Capture email thread

To add an email message (.msg) to a bug

  1. Open the thread you want to save (or, select the message in your Inbox).
  2. Select the File tab on the Outlook ribbon, and then click Save As.
  3. In the Save As dialog, select a temporary storage area and make sure the type is set to the default – Unicode .msg file.
  4. Name and save the file.
  5. Open the bug.
  6. Select the Attachments tab.
  7. Click Add New.
  8. Navigate to the .msg file you created above.
  9. Select the .msg file and then click Save.

Close a bug

To close a bug

  1. Find the bug.
  2. Select the bug number, right-click, and then select Edit Work Item.
  3. Fill in or select the value for the following fields:

  5. Open the bug again and then save the State to Closed.
