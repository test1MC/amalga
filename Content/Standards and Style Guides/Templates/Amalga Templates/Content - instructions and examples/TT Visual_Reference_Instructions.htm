Access

For visual reference topics, document only the access method that is best from a usability standpoint.
• Preferred: If there is a topic that provides all needed steps to access a screen using a specific path, xref that topic.
• Alternative: If there is no topic that provides all needed steps, then do the following.
       ~ If the UI element is inside Management Console, xref the Open Mgmt Console proc.
        ~ If the steps needed to get to the screen are 3 or more, write an access topic, and xref that topic.
         ~ If there are two or fewer steps, put the steps in this topic (inserted as individual step snippets for reuse). Reuse the step snippets in related task topics so that all versions stay in sync.
