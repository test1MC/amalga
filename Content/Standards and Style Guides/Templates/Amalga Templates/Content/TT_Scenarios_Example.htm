Here are a sseries of et o procedures get you started with .

Create a list

  1. First, you need to create a patient list. There are several types of criteria you can use. See Create a list using criteria.
  2. If you want to use more complex logic to create a list, see Create a list by combining criteria.
  3. Once your list is created, you can view it. See View a list.
  4. You can view reports in a graphical form along with the associated list. See View the pane.
  5. If there is no report available for the list, click Start Report Builder to add a report. See Create a report.

View a list

  1. On the Patient Lists page, you can view any of the lists to which you have access. See View a list.
  2. Then you can view reports associated with that list. See View the pane.
  3. To return to the simple grid view of the list, see View a patient list in grid mode.
  4. From the list, you can pick a specific patient and view their patient chart. See Start .
