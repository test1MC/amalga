Creating a new generic control

 

This topic describes how to create a new Generic control instance.

You can use the Generic controls to build a new control from scratch. Generic control instances can inherit from either the BaseServerControl class or the BaseUserControl class.

In this topic

  • Creating a new control
  • Adding functionality to the control

Creating a new control

You can create a new generic control instance by first creating a project using the C# Class Library template, and adding the Server Control template. You can then modify the control to meet your needs. The following steps describe how to create a new Server Control project.

  1. In , create a new project using the C# Class Library template.
  2. Delete the default class (named Class1.cs).
  3. Right-click the Class Library project node.
  4. Click Add new item.
  5. Under Installed Templates select Amalga.
  6. Select either Amalga Server Control or Amalga User Control, and click Add.
