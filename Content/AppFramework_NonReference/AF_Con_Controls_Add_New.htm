Create a new control project

The following steps show how to create a new control project.

  1. In the Visual Studio File menu, click New, then click Project.
  2. In the New Project dialog box, in the Installed Templates pane, expand the Visual C# node and select Amalga.
  3. In the list of templates, select Amalga Empty SP Project.
  4. In the drop-down menu above the list of templates, select .NET Framework 3.5.
  5. Click OK to create the project.
  6. In the SharePoint Customization Wizard dialog, enter the URL pointing to your default staff portal site. For example: https://server.contoso.com:8080/staff/
  7. Select the Deploy as a farm solution option.
  8. Click Finish to create the project.
  9. In Solution Explorer, select the project node.
  10. On the Project menu, click Add New Item.
  11. In the Add New Item dialog box, in the Installed Templates pane, expand the Visual C# node, and then click Amalga.
  12. In the list of SharePoint templates, select Amalga User Control.
  13. Type a name for the user control in the Name box, and then click Add.

Add code to the control

The base control project contains all of the functionality that is required to interact with Amalga.
