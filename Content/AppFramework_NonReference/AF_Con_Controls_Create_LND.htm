Create and extend controls

 

This topic covers all of the available options for creating and extending controls.

There are multiple paths to creating a fully instrumented Amalga control. You can extend the functionality of an existing control, create new custom controls by deriving from the generic control classes, and wrap an existing control. Each of these patterns will result in controls that are compatible with the client framework, and capable of context passing, error handling, logging, notifications, and other features.
