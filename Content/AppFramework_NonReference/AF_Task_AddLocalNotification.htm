Add a local notification

This topic describes how to add an local notification to a web page.

A local notification is a structured message that provides immediate feedback in response to user activity, or alerts the user to application activity. Local notifications are not persistent, are not managed by the Notification web service, and are not available outside the current web page.

Local notification content

A local notification can display any string value. The content may include HTML tags for embedded hyperlinks and formatting.

It is a good practice to define the string in a resource (.resx) file. There is no direct support for parameterized values, but any string handling helper method can be used as a preprocessor for the notification content. The example below shows how the parameterized string support in the LocalizationManager helper methods can be used to add localized, parameterized string values to a local notification.

 

Example

The following C# snippet uses the NotificationManager and LocalizationManager to add a parameterized local notification to the current web page.

// NOTE: in this example, a resource string has been added in advance 
//  to MyApp.resx, a resource file in the App_GlobalResources folder 
//  of the default SharePoint IIS site.  The resource name is 
//  "myLocalNotificationResource" and its value is
//  "<b>Warning:</b><br/>{0}".

// Add a local notification to the current page.
//  Note that local notifications, unlike user notifications, 
//  do not require type definition or registration.
AddLocalNotification(
    this.Page, 				            // target the current page
    LocalNotificationType.warning,      // severity - warning
    LocalizationManager.GetResourceValue(
        "MyApp.resx",                        // resource file name
        "myLocalNotificationResource",       // resource key name
        new string[] {"This is a warning message."}  // message text
    ),
    true, 				// allow user to close
    false				// do not add to the top of existing notifications
);
