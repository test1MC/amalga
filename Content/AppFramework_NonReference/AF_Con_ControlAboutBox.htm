About Box

 

The About Box is used to display information about the platform to end-users.

The About Box control is not intended to be extensible by partners. Third-party developers must implement their own "about box," and may optionally launch the About Box control from within their control. The About Box provides information relative to the platform, including the name of the application, its version number, the base version build number, copyright information, all applicable third party attributions, and a feedback link. By default the About Box can be accessed from portal page footers.

Invoking the About Box

You can invoke the About Box control by using the following URL:

https://[yourURL]:8080/_layouts/amalga/about.aspx

To ensure that the About Box is displayed properly, we recommend using something similar to the following HTML:

<a href="#" onClick="MyWindow=window.open(

'https://[yourURL]:8080/_layouts/amalga/about.aspx',

'MyWindow','toolbar=no,

location=no,

directories=no,

status=no,

menubar=no,

scrollbars=no,

resizable=no,

width=800,

height=520'); return false;">About </a>

The sample HTML displays the About Box in a new non-resizable 800x520 pixel browser window, with no scroll bars and no address bar.
