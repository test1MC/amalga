Retrieving data using the

This section describes thedata access features available to applications.

The SDK provides several data access APIs. These APIs allow hosted applications, including web applications, to access the data provided by application views.

All of the data access APIs work by making SQL queries against an using the . They differ in the methods they use to establish a data connection, and in the ways they expose the available data. Several of the APIs implement the standard data access patterns provided by and ADO.NET.
