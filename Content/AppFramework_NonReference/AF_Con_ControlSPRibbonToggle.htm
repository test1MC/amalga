Ribbon Toggle control

 

The Ribbon Toggle control adds the ability to turn the standard Ribbon on and off.

The Ribbon Toggle control exposes standard functionality, by enabling users to turn on the Ribbon. You can use the Ribbon to configure and manage your site. To toggle the ribbon on any of the default portal pages, click the caret ("^") symbol in the upper-right corner of the page.

Adding the Ribbon Toggle control to a page

You can add a Ribbon Toggle control to a project in by following the instructions in Toolbox controls.

Controlling access to the Ribbon Toggle control

The Ribbon Toggle control is only visible to users who have the appropriate permissions. The control uses the SPSecurityTrimmedControl to determine if the toggle link should be displayed. An example of this is shown in the following example.

<Sharepoint:SPSecurityTrimmedControl ID="SPSecurityTrimmedControlForRibbon" runat="server" PermissionsString="AddAndCustomizePages">

In the preceding example, the ribbon will be hidden from all users that do not have "AddAndCustomizePages" privileges. The default permissions are defined as follows:

Users with "Design" or greater permissions can access the following features:

  • Change Privileges
  • Add users
  • Modify site navigation configuration
  • Modify site or page properties

Users with "Contribute" or greater permissions can access the following features:

  • Upload document
  • Add a folder (to a doc library)
  • Add a wiki page
  • Modify document metadata/properties

Although uses only the default ribbon functionality, you can further extend the ribbon by using the API. For more information, see Server Ribbon in Foundation.
