Add legal content

 

This topic describes the types of legal content that can be presented on the site and the form that it can take.

provides legal content in the form of footer links, in-page links, and a pop-up template. The and each have their own set of links, which you must configure. Each link opens a new browser window using the supplied generic pop-up template. Alternatively, you can link to an external URL.

The following table describes the types of legal content that you can customize.
