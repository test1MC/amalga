Verify application installation

You can use the following procedures to verify application installations.

Data connector

To verify the data connection application installation

  1. Open a browser and access the following data connector URL using the fully qualified domain name of the Application host.

    https://<FQDN>:8801/DataConnectorService/Microsoft.Amalga.Data.DataConnectorWCFWS.DataConnectorService.svc

  2. If you can browse to the URL, the data connector is operational.
