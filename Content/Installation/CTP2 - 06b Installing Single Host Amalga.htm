Installing in the single host environment

After Once you have completed the prerequisite steps for a single host environment, continue with the installation.

Download and run the installation script

The installation script for a single host environment is not included with your Amalga distribution CD.

  1. Go to http://connect.microsoft.com and navigate to the download page. Your sign-in to this site is part of the invitation to participate in program.
  2. Download the SamplesAndTools.zip file and save it to a folder on your desktop called 'AmalgaSingleInstall'.
  3. Unzip the contents of the download
  4. From the extracted files, open Single-boxInstall.zip (contained in SamplesAndTools.zip) and follow the instructions in the InstallAmalgaReadme.doc file
