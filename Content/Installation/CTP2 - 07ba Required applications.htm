Required Applications

To create a functioning system, the following applications need to be installed in the order listed and on the server specified. The server names correlate to the diagram following the table.

The Data Ingestion, Imaging, and Data Explorer applications must be installed according to separate documentation. For those applications, use the associated links to access the installation procedures. If the documentation column contains "NA", the application can be installed with the documentation in this guide by starting with Application manifest resolution.

Read the important application notes below before beginning installation.

  • Although other documentation must be used to install the Data Ingestion, Imaging, and Data Explorer applications, they must still be installed in the order listed.
  • Although other documentation must be used to install the Data Ingestion, Imaging, and Data Explorer applications, be sure to consult the Configure applications section to complete the database access configuration and other steps as required.
