Network DTC configuration

 

Network DTC must be installed and activated on each server in the topology prior to installation.

For multi-box implementations, network DTC must be installed and configured as specified in this section on the server hosting the ClinicalData database.

To configure network DTC

  1. Access the network DTC properties as specified in Enable Network DTC Access (http://technet.microsoft.com/en-us/library/cc753510(WS.10).aspx).
