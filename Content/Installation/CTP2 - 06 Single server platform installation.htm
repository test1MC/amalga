Single server host installation

 

This section describes how to install in a single host environment. There are two basics steps for this process:

Prepare the host

Use the installation script to complete the installation.

To install on a single server

  1. Go to http://connect.microsoft.com and navigate to the download page. Your sign-in to this site is part of the invitation to participate in program.
  2. Download the SamplesAndTools.zip file and unzip the contents.
  3. Open Single-boxInstall.zip (contained in SamplesAndTools.zip) and follow the instructions in the InstallAmalgaReadme.doc file.
