Installing

Complete the preparation steps for 'All Servers', 'Platform', 'Database', and 'Application' before attempting the following steps.

For a full list of command arguments see Setup utility reference.

 

On the Platform Host

Install Platform Services

  1. Open a Doors Command prompt with Administrator permissions
  2. Navigate to the AmalgaInstallationFiles folder you created earlier
  3. Navigate to the 'Platform' folder (EXAMPLE. c:\users\myuser\desktop\AmalgaInstallationFiles\Platform)
  4. Type the following command at the prompt:
    Setup.exe /Feature=PlatformServices /Install /LogicalHostName=<NameofPlatormLogicalHost> /SystemDBServerInstance=<NameofDatabaseLogicalHost> /PlatformClusterName=<FullyQualifiedDomainNameofPlatformHost> /PlatformSSLThumbPrint=<ThumbPrintofSSLCertificate> /LogicalHostType=NONE /eula accept

EXAMPLE: Setup.exe /Feature=PlatformServices /Install /LogicalHostName=AmalgaPlatform /SystemDBServerInstance=AmalgaDatabase /PlatformClusterName=AmalgaPlatform.ContosoHospital.com /PlatformSSLThumbPrint=a12b3c4e5f6a7b8c9e0f1a2b3c4e5f6a7b8c9dff /LogicalHostType=NONE /eula accept

Install

  1. Open a Doors Command prompt with Administrator permissions
  2. Navigate to the AmalgaInstallationFiles folder you created earlier
  3. Navigate to the 'Platform' folder (EXAMPLE. c:\users\myuser\desktop\AmalgaInstallationFiles\Platform)
  4. Type the following command at the prompt:
    setup.exe /Feature=ManagementConsole /Install /PlatformClusterName=<FullyQualifiedDomainNameofPlatformCluster> /LogicalHostName=<NameofPlatformLogicalHost> /AdminWebsiteSSLThumbprint=<ThumbPrintofSSLCertificate> /AdminWebsiteClusterName=<FullyQualifiedDomainNameofPlatformHost> /eula accept

EXAMPLE: setup.exe /Feature=ManagementConsole /Install /PlatformClusterName=AmalgaPlatform.ContosoHospital.com /LogicalHostName=AmalgaPlatform /AdminWebsiteSSLThumbprint=a12b3c4e5f6a7b8c9e0f1a2b3c4e5f6a7b8c9dff /AdminWebsiteClusterName=AmalgaPlatform.ContosoHospital.com /eula accept

Install Storage

  1. Open a Doors Command prompt with Administrator permissions
  2. Navigate to the AmalgaInstallationFiles folder you created earlier
  3. Navigate to the 'Platform' folder (EXAMPLE. c:\users\myuser\desktop\AmalgaInstallationFiles\Platform)
  4. Type the following command at the prompt:
    Setup.exe /Feature=DataPipelineStorage /Install /TerminologyDBName=<NameofTerminologyDatabase> /ClinicalDataDBServerInstance=<NameofDatabaseServer> /MessageQueueDBServerInstance=<NameofDatabaseServer> /eula accept

EXAMPLE: Setup.exe /Feature=DataPipelineStorage /Install /TerminologyDBName=HLI /ClinicalDataDBServerInstance=AmalgaDatabase.ContosoHospital.com /MessageQueueDBServerInstance=AmalgaDatabase.ContosoHospital.com /eula accept

Install Feature Application Services

  1. Open a Doors Command prompt with Administrator permissions
  2. Navigate to the AmalgaInstallationFiles folder you created earlier
  3. Navigate to the 'Platform' folder (EXAMPLE. c:\users\myuser\desktop\AmalgaInstallationFiles\Platform)
  4. Type the following command at the prompt:
    Setup.exe /Feature=ApplicationServices /Install /LogicalHostName=<NameofPlatformLogicalHost> /LogicalHostType=None /ApplicationSSLThumbPrint=<ThumbPrintofSSLCertificate> /PlatformClusterName=<FullyQualifiedDomainNameofPlatformClusterHost> /ApplicationClusterName=<FullyQualifiedDomainNameofPlatformClusterHost> /eula accept

EXAMPLE: Setup.exe /Feature=ApplicationServices /Install /LogicalHostName=AmalgaDatabase /LogicalHostType=None /ApplicationSSLThumbPrint=a12b3c4e5f6a7b8c9e0f1a2b3c4e5f6a7b8c9dff /PlatformClusterName=AmalgaDatabase.ContosoHospital.com /ApplicationClusterName=AmalgaPlatform.ContosoHospital.com /eula accept
