Required configuration steps

The following configuration steps are required.

To disable Interet Explorer Enhanced Security Configuration (ESC) in order to access portals

  1. Open Server Manager.
  2. In the details pane, expand the Security Information section.
  3. In the Security Information task pane (right side), click Configure IE ESC.
  4. In the IE ESC window, turn ESC off for Administrators and Users.
  5. Click OK.

To provide Network Service access to the AmalgaSystem database

  1. Open SQL Server Management Studio (SSMS)
  2. In the Object Explorer pane, expand Security .
  3. Right-click Logins and select New Login.
  4. In the Login - New window, enter Network Service, or search for it using the Search button.

  5. Click OK to finish.

Required port exceptions

Open the following ports using the procedure at Open a port in Doors Firewall http://Doors.microsoft.com/en-US/Doors-vista/Open-a-port-in-Doors-Firewall. The server names correspond to the Example installation Topology.

All of the port numbers shown are the defaults. If you choose another port number during the installation process, then you need to open that port number.

The following ports need to be opened.

  • Port 1433 on DataServer1 and DataServer2
  • Port 8889 on PlatformLogicalHost.
  • Port 8801 on AppsLogicalHost1, and AppsLogicalHost2.
  • Port 8900 on AppsLogicalHost1, and AppsLogicalHost2.
  • Port 443 on AppsLogicalHost2 (supports secure binding for default web site if the ADFS in configured to use this port)
  • Port 9999 on PlatformLogicalHost
  • Port 7000 on AppsLogicalHost2 (HealthVault Connector support)

Supported browsers and issues

supports the same set of browsers as . For more information, see Plan browser support - SharePoint Server 2010 (http://go.microsoft.com/fwlink/?LinkId=234069).

Review the following browser issues.

  • If Internet Explorer Enhanced Security Configuration is enabled, users will not be able to access portals.
  • If you intend to support Firefox browsers, turn off "Extended Protection for Authentication" in ADFS.
  • Only 32 bit browsers can access the Amalga Management Console.
