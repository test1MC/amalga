Primary documentation access

You can find the most recent version of this document and a complete set of documentation on the Community Portal for Microsoft's health customers by using the steps below.

To open documentation

  1. Open the portal at http://customers.mshealthcommunity.com.
  2. Click Sign In and enter your credentials.
  3. If you need to acquire credentials to enter the site, request access at the following location: http://customers.mshealthcommunity.com/user/InviteUser.aspx.
  4. Navigate to the page.
  5. Click Documentation.
