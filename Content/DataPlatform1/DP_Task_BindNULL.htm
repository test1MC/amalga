Configure binding: NULL values

 

This topic describes how to bind entity properties to NULL values.

You can bind entity properties to NULL values when the properties are not required by any applications that are bound to the entity. The related application views indicate lack of requirement for properties by either marking the properties as optional or not referencing the properties at all.


Required Role: AmalgaDataAdmin or AmalgaSysAdmin

To bind a property to the NULL value

  7. For the desired entity property, select NULL from the drop-down list of binding options.
