Configure binding: constant values

 

This topic describes how to bind entity properties to constant values.

You can bind entity properties to constant values. Binding properties to constant values allows you to meet mandatory binding requirements when NULL values are not allowed. For example, binding is mandatory for all properties of an entity if the entity is bound to an application and the corresponding application view columns are not explicitly declared as optional.


Required Role: AmalgaDataAdmin or AmalgaSysAdmin

To bind a property to a constant value

  7. For the desired entity property, select the constant value entry option () from the drop-down list of binding options and then enter the constant value that you want.
