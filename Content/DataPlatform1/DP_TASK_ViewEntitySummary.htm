View an entity's bound applications and access policy

 

This topic describes how to view the entity summary pane for an entity.

You can view an entity's bound applications, access policy summary, and list of properties in the entity summary pane. You can also view details about each entity property, such as applied semantic tags.


Required Role: AmalgaDataAdmin or AmalgaSysAdmin (AmalgaAppAdmin suffices for viewing bound applications and entity properties)
