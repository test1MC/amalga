Configure binding: selected column ( tab)

 

This topic describes how to bind entity properties to columns selected from the storage schema listed in the tab.

You can bind entity properties to columns selected from storage schema.

For information about defining entity bindings using , see Set-AmEntityBinding.


Required Role: AmalgaDataAdmin or AmalgaSysAdmin

To bind a property to a column selected from storage schema
