Configure binding: recommended column

 

This topic describes how to bind entity properties to recommended columns.

You can bind entity properties to recommended columns. Columns for binding are recommended for each property when an anchor table exists for the entity (in the current binding source) and when columns from the anchor table match one or more semantic tags applied to the properties.


Required Role: AmalgaDataAdmin or AmalgaSysAdmin

To bind a property to a recommended column

  7. For the desired entity property, select a recommended column from the drop-down list of binding options.
