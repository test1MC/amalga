Find entities in the registry

 

This topic describes the filtering capability in the .

You can filter the list of entities in the registry by application-binding status. You can show only those entities that are bound to one or more applications, bound to a specific application, or not bound to any applications.


Required Role: AmalgaAppAdmin, AmalgaDataAdmin, or AmalgaSysAdmin

To filter the list of entities in the registry

  2. In the text box above the folder in the navigation pane, select a filter option.

    The navigation pane is refreshed; if the folder was expanded, it contracts.

  3. Expand the folder.

    The navigation pane lists namespaces and entities that match the selected filter option.

  4. To remove the filters and redisplay all entities in the registry, select All from the text box.
