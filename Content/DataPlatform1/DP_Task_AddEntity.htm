Create entities

 

This topic describes how to add entities to the .

You can extend the data model or develop a new data model by creating entities. For example, a new application may require data not yet referenced by the . In such a case, new entities are needed.

To access data through a new entity, you must bind the entity to storage schema and define access policies for desired roles. For more information, see Entity binding overview and Grant permissions to an entity (add an access rule).
