Microsoft.Terminology.Relationship entity

 

Attribute Description
LanguageId Unique identifier of the language associated with the Meaning attribute in the Microsoft.Terminology.Concept entity.
RelationshipTypeId Unique identifier of the relationship type, which is a foreign key to the Microsoft.Terminology.RelationshipType entity.
RelationshipTypeName Description of the relationship type.
SourceconceptCode For a binary relationship, the source concept code defined by the source terminology. For a unary relationship, the concept code.
SourceconceptCodeId Unique identifier of the source concept code, which is a foreign key to the CodeId attribute in the Microsoft.Terminology.Concept entity.
SourceConceptId Unique identifier for the source concept, which is a foreign key to the ConceptId attribute in the Microsoft.Terminology.Concept entity.
SourceConceptMeaning Description of the source concept.
TargetConceptCode For a binary relationship, the target concept code defined by the target terminology.
TargetConceptCodeId Unique identifier of the target concept code, which is a foreign key to the CodeId attribute in the Microsoft.Terminology.Concept entity.
TargetConceptId Unique identifier of the target concept, which is a foreign key to the ConceptId attribute in the Microsoft.Terminology.Concept entity.
SourceConceptMeaning Description of the target concept.
TerminologyId Unique identifier of the terminology that contains the current relationship.
