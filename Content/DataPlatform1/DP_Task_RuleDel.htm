Delete an access rule

 

This topic describes how to delete access rules in access policies for entities.

You can delete access rules. Deleting an access rule modifies the data visible for the indicated role.


Required Role: AmalgaDataAdmin or AmalgaSysAdmin

To delete an access rule

  4. In the Authorized Roles section, click a role to display its access rules.
  5. In the Rules applying to Role section, expand the group that contains the access rule you want, click the access rule, and then click Delete Rule.
