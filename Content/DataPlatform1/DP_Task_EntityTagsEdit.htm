Apply semantic tags to entities

 

This topic describes how to change the semantic tags applied to entities and their properties.

You can apply and remove semantic tags at the level of an entity or of one of its properties.

Only entities in local namespaces can be edited.

To change the semantic tags applied to an entity or entity property

  3. .

  4. To change the semantic tags applied to the entity, click in the Entity Name section.
  5. To change the semantic tags applied to an entity property, select the entity property and then click in the Property Details section.

  6. In the dialog box, do the following.

    To... Do this: 
    Search for a semantic tag by description ("Meaning" property of the concept) Select and then type part or all of the description.
    Search for a semantic tag by code ("Code" property of the concept) Select and then type part or all of the code.
    Apply a semantic tag Select the tag in the Search Results section and then click Add.
    Remove a semantic tag Select the tag in the Selected Tags section and then click Remove.
  7. Click OK to save changes and close the dialog box.
  8. Click Save Entity to save changes and close the dialog box.
