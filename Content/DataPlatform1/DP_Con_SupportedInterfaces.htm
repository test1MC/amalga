Supported interfaces

 

This topic lists the interfaces supported for secure data access.

Data access is secure when roles consume data from supported interfaces.

Supported interfaces for secure data access: 

  • R/W ADO.NET client for the data connector
  • OLE DB client for the data connector, supporting such clients as , , and (SSRS)
  • data source for ASP.NET and Sharepoint 2010
  • Imaging metadata
  • Data Change Notification (DCN)
  • External connector (and other external systems)
