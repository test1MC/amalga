Unpack Materializer CAB file

 

To locate and unpack Materialization manifest XML files

  1. Create a staging folder with a nested temporary folder. For example, create C:\Materialize Manifest to hold the CAB files and C:\Materialize Manifest\MasterConfig to hold the extracted XML files.
  2. Navigate to root\v3\Rel\Applications\Release and locate Materialize.CAB.
  3. Copy the set of CAB files to the staging folder.
  4. Extract the XML files to the \MasterConfig subfolder. Keep the extracted XML files as master configuration files from which you can generate application instances.
  5. Create a subfolder for each new entity. For example, create C:\Materialize Manifest\Patient for the Patient entity.
  6. Copy a full set of extracted XML configuration files to the entity folder. For example, copy the XML files from the master folder (C:\Materialize Manifest \MasterConfig) to the entity folder (C:\Materialize Manifest\Patient).
