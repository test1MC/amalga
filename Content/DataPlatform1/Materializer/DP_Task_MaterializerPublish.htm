Publish an application instance

 

For example, type the following command to publish an instance of the Materializer application using the specified CAB and auxiliary files:

Publish-ApplicationInstance -ApplicationCabFile C:\Manifest\Materializer.cab -ApplicationAuxiliaryFiles "C:\Manifest\MaterializerApplication_Patient.xml;C:\Manifest\MaterializerService_Patient.xml"

  1. Go to the next procedure: Deploy an application instance.
