Grant permissions to sensitive data

 

This topic describes how to grant permissions for entity properties to which sensitive labels are applied.

You can grant permissions to roles for sensitive labels. Granting permissions for sensitive labels ensures that the role can access the underlying data in the entity properties to which the sensitive labels are applied. See Restrict access to an entity property (sensitive labels) for more information.


Required Role: AmalgaDataAdmin or AmalgaSysAdmin
