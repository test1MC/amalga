Edit an entity relationship (access inheritance)

 

This topic describes how to edit entity relationships that determine access inheritance.

You can edit entity relationships by changing the entity property selected for mapping to the related entity.

If you want to change the relationship type, remove the entity relationship and then add it again with the relationship type you want. You can remove an entity relationship by selecting its check box and then clicking .
