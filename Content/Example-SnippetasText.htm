Once you have your plain paragraph, you can insert the snippet. Convert it to text as shown in step 4 in the following procedure.

You can insert snippets and then convert them to plain text. This method is useful for template-style snippets, such as snippets containing empty tables that you want to modify.

In keeping with best practices for single sourcing, reserve the "insert as text" snippet action for template-style snippets. Other snippets should be retained as snippets so that any future changes can be done at the snippet level for automatic updates to any topics using the snippets.

To insert a snippet as text

  1. In your topic, press ENTER to create an empty paragraph where you want to insert the snippet. (Unless you are inserting a short snippet that you want to be part of an existing line, in which case, place your cursor in a paragraph that already contains characters.)

    2. To create an empty paragraph in a list (for example, to insert a snippet of a table), after pressing ENTER to get the new <li> block in the list, right-click the <li> block and select Make Paragraph Item(s), and then right-click the new <p> block and select Move Left.

  2. Make sure the cursor is at the position where you want the snippet to be inserted.
  3. Select Insert > Snippet and select the desired snippet file.

    4. The inserted snippet is indicated by a new "MadCap:snippet" block.

  4. Right-click the inserted snippet and select Convert to text.

    5. The inserted snippet is now converted to text and ready for your updates, as indicated by the change from the "MadCap:snippet" block to an HTML block.
