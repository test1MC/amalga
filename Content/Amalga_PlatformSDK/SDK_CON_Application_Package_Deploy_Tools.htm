Application publishing and packaging tools

This topic describes the application publishing and packaging tools available for hosted applications.

provides a publishing feature that you can use to publish hosted applications for debugging purposes from within the integrated development environment (IDE).

The publishing feature allows you to rapidly test a hosted application:

  • Automatically removes existing instances of the hosted application.
  • Automatically resolves the application manifest and service manifests by using testing-specific elements and attributes in the application manifest and service manifests of a hosted application.
  • Deploys the hosted application on a specified target topology.

For more information about using the application publishing tools to debug a hosted application in , see Debugging a hosted application on page 1.

Once you have finished testing, you can use the packaging feature (also provided by ) to package your hosted application into an appropriately structured file for deployment using the deployment tools provided with the  platform.

For more information about using the application packaging tools to package a hosted application in , see Deploying hosted applications on page 1.
