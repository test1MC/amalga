Source systems

 

This topic describes source systems and how the Terminology platform application interacts with them for semantic tagging purposes.

In data ingestion terms, a source system identifies a specific incoming data feed. The Terminology platform application can map a source system to a clinical terminology for data ingestion purposes, allowing clinical concepts in incoming messages to be consistently and accurately identified when messages from the source system are shredded.

Messages submitted by the source system to the data pipeline are examined by the Message Shredder subsystem, which applies semantic tags to the columns it shreds automatically when the message type supports semantic tags. For more information about the subsystem, see Message Shredder application on page 1.

To ensure that a message type can support semantic tags, the Terminology platform application stores metadata about the source system, mapping the data feed provider and message type to a specific clinical terminology, so that the Message Shredder subsystem can identify and, if needed, transcode clinical concepts to incoming columns.

You can programmatically add a new source system to the Terminology platform application, by using the ISemanticLabelTaggingService service contract from the TerminologyService service. For more information about adding source systems, see Adding source systems on page 1. Once added, you can retrieve information about existing source systems, also by using the ISemanticLabelTaggingService service contract. For more information about querying source systems, see Querying source systems on page 1

You can also deactivate semantic tagging for an existing source system by using the same service. Deactivating the source system does not delete the mapping from the Terminology platform application, but instead indicates that the pipeline application should not use that mapping during message shredding. For more information about deactivating a source system, see Deactivating source systems on page 1.
