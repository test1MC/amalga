Synonyms

 

This topic describes synonyms, which are concepts identified using mapping terminologies that represent the same clinical idea across clinical terminologies.

A synonym is a concept in one clinical terminology that is an acceptable alternative to a concept in a different clinical terminology, as identified by a mapping terminology. The same clinical concepts can be described in very different ways between clinical terminologies. Synonyms allow different representations of the same clinical concept to be mapped to each other, providing transcoding functionality when ingesting data. Synonyms are not necessarily unique; that is, more than one clinical concept may use the same synonym.

Synonyms are identified by a mapping terminology, used to map concepts between clinical terminologies. You can discover synonyms by using either the ITerminologyQueryService service contract or the Microsoft.Terminology.Beta.Synonym entity. For more information about querying synonyms, see Querying synonyms on page 1.
