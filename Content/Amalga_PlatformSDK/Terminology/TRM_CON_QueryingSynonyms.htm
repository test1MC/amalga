Querying synonyms

 

This topic describes how to retrieve information for synonyms identified in and stored by the Terminology hosted application.

You can use either the ITerminologyQueryService service contract, from the TerminologyService service, or the Microsoft.Terminology.Beta.Synonym entity to retrieve information about synonyms from the Terminology hosted application.

In this topic

  • Using ITerminologyQueryService on page 1

Using ITerminologyQueryService

You can use the GetSynonyms operation, included with the ITerminologyQueryService service contract, to retrieve an array of Concept objects, in which each Concept object represents a synonym, given a specified concept ID.

Example

The following example uses the GetSynonyms operation from the ITerminologyQueryService to retrieve synonyms associated with the clinical concept for hypertensive disease in the ICD-9 clinical terminology.

public Concept[] GetSynonymsFromService()
{
    // Use GetSynonyms from ITerminologyQueryService to retrieve
    // an array of Concept objects.

    // First, resolve the service endpoint and connect to the service.
    // Resolve the service endpoint.
    ServiceEndPoint terminologyEndPoint = HostedServiceResolver.ResolveServiceEndPoint(
       (ApplicationDependency)ApplicationConfiguration.Dependencies["TerminologyApplicationDependency"],
       new Guid("8c3be61d-021b-419a-ad9a-4c74b7b115ea"),
       "Microsoft.Amalga.Terminology.ServiceContracts.ITerminologyQueryService");

    // Open the client channel and instantiate the WCF client.
    ITerminologyQueryService terminologyServiceClient =
       AmalgaWcfChannelFactory<ITerminologyQueryService>.CreateChannel(terminologyEndPoint);

    // Identify the concept for which synonyms are to be retrieved. The
    // concept identifier provided below describes the clinical concept for
    // hypertensive disease in the ICD-9 clinical terminology.
    string conceptID = "5794577";

    // Invoke the operation from the service, and
    // return the resulting array of Concept objects.
    return (Concept[])terminologyServiceClient.GetSynonyms(conceptID);
}

Using Microsoft.Terminology.Beta.Synonym

You can also use the Synonym entity, included in the Microsoft.Terminology.Beta entity namespace, to retrieve synonyms. Once an application view for the Synonym entity is added to the virtual schema of your hosted application, you can then use to retrieve information from the application view. For more information about using to retrieve information from application views, see Retrieving data using AmalgaDbDataReader on page 1.
