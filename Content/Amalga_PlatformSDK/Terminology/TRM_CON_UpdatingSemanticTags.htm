Updating semantic tags

 

This topic describes how to update semantic tags for an existing positional data vector in the Terminology platform application.

You can use the UpdateSemanticLabel operation, included in the ISemanticLabelTaggingService service contract, to add a semantic tag to an existing positional data vector. The operation accepts two parameters:

  • positionalDataVectorId

    A System.Guid object representing the globally unique identifier (GUID) that identifies an existing positional data vector.

    This identifier can be discovered by using the GetPositionalDataVectors operation from the ISemanticLabelTaggingService service contract. For more information about querying positional data vectors, see Querying positional data vectors on page 1.

  • semanticConceptGuid

    A string value representing the GUID that identifies an existing concept.

    This identifier can be discovered by using the GetConcepts operation from the ITerminologyQueryService service contract, or by querying the Microsoft.Terminology.Beta.Concept entity. For more information about querying concepts, see Querying concepts on page 1.

No more than 64 semantic tags can be applied to a single positional data vector.
