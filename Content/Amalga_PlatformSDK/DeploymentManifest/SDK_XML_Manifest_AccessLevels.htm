AccessLevels element

 

This topic describes the AccessLevels element.

Declares the different access levels exposed by the hosted application.

Syntax

<ApplicationAuthorization>
   <AccessLevels>
      <AccessLevel>...</AccessLevel>
   </AccessLevels>
</ApplicationAuthorization>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 1-1: Required element that can occur once and only once.

Element relationships

Relationship Element
Parent elements ApplicationAuthorization
Child elements AccessLevel

Remarks

The access levels defined in the AccessLevels element are used in when specifing to a user role a specific access level for the hosted application.
