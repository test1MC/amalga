MinVersion element

 

This topic describes the MinVersion element.

Specifies the minimum version of the dependency required for the hosted application.

Syntax

<Dependency>
   <MinVersion>string</MinVersion>
</Dependency>

Element characteristics

Characteristic Description
Data type and length String, representing a System.Version object.
Default value None
Cardinality 1-1: Required element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Dependency
Child elements None

Remarks

The value of this element must be set to a valid string representation of the major, minor, build, and revision components of a System.Version object.
