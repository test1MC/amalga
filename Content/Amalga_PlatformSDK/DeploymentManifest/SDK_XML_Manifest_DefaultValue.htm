DefaultValue element

 

This topic describes the DefaultValue element.

Declares the default value for the configuration setting.

Syntax

<Setting>
<DefaultValue>
string | <CDATA>
   </DefaultValue>
</Setting>

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Setting
Child elements None

Remarks

When the hosted application is first deployed, the value specified in this element is used as the default value for the configuration setting represented by the Setting parent element. This value is also used by default while debugging the hosted application. After deployment, the value specified for the configuration setting can be changed with a variety of methods. For more information about modifying the value of a configuration setting after deployment, see Setting Element on page 1.
