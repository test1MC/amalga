Settings element

 

This topic describes the Settings element.

Declares a set of configuration settings that, when the hosted application is deployed, are stored in the configuration store for the object represented by the deployment manifest.

Syntax

<DeploymentManifest>
   [ <Settings>
      <Setting>...</Setting>
   </Settings> ]
</DeploymentManifest>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements DeploymentManifest
Child elements Setting

Remarks

The Settings element contains the configuration settings that are displayed in for the object represented by the deployment attribute after a hosted application has been deployed.
