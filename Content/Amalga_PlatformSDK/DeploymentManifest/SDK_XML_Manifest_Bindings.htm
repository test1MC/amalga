Bindings element

 

This topic describes the Bindings element.

Declares a set of bindings for a service contract in a hosted .

Syntax

<Contract>
   <Bindings>
      [ <FederationBinding>...</FederationBinding> ]
      [ <QueueBinding>...</QueueBinding> ]
   </Bindings>
</Contract>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 1-1: Required element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Contract
Child elements FederationBinding, QueueBinding

Remarks

This element contains the collection of Binding elements that apply to the service contract described by the parent Contract element.
