Deployment manifest XML hierarchy

 

This topic describes the hierarchy of XML elements included in a deployment manifest.

The following XML hierarchy describes the XML elements used by deployment manifests.

<DeploymentManifest>
<Settings>
<Setting>
<DefaultValue>
<Validation>
<ApplicationManifest>
<ApplicationAuthorization>
<AccessLevels>
<AccessLevel>
<Role>
<Resources>
<Resource>
<AccessLevel>

<Dependencies>
<Dependency>
<Id>
<MinVersion>
<TestConfiguration>
<WcfServiceManifest>
<Contracts>
<Contract>
<Bindings>
<FederationBinding>
<QueueBinding>
<DoorsWorkerManifest>
<PerformanceCategories>
<PerformanceCategory>
<PerformanceCounter>
