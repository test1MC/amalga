Contracts element

 

This topic describes the Contracts element.

Declares the service contracts applicable to the hosted WCF service.

Syntax

<WcfServiceManifest>
   <Contracts
      HostAssembly = "string"
      ServiceClass = "string" >
      <Contract>...</Contract>
   </Contracts>
</WcfServiceManifest>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 1-1: Required element that can occur once and only once.

Element relationships

Relationship Element
Parent elements WcfServiceManifest
Child elements Contract

Attributes

Attribute Description
HostAssembly

(Required string.) The name of the assembly in which the service contracts are implemented.
ServiceClass

(Required string.) The fully qualified name of the class that represents the hosted WCF service. For example, "Namespace.Application.ServiceClass".
