Contract element

 

This topic describes the Contract element.

Declares a service contract exposed by a hosted .

Syntax

<Contracts>
   <Contract Name = "string" >
      <Bindings>...</Bindings>
   </Contract>
</Contracts>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 1-n: Required element that can occur one or more times.

Element relationships

Relationship Element
Parent elements Contracts
Child elements Bindings

Attributes

Attribute Description
Name

(Required string.) The fully qualified name of the object or member exposed by the service contract. For example, "Namespace.Application.ServiceInterface".

Remarks

The Contract element represents a service contract, which determines what objects or members are exposed for use by other hosted applications, as well as the bindings that apply to that service contract.
