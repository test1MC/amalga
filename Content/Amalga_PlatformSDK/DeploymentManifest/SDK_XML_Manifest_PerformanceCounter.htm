PerformanceCounter element

 

This topic describes the PerformanceCounter element.

Declares a single performance counter for a or hosted .

Syntax

<PerformanceCategory>
   <PerformanceCounter
      Name = "string"
      CodeName = "string"
      Help = "string"
      Type = "string" />
</PerformanceCategory>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 1-n: Required element that can occur one or more times.

Element relationships

Relationship Element
Parent elements PerformanceCategory
Child elements None

Attributes

Attribute Description
Name

(Required string.) The display name of the performance counter.
CodeName

(Required string.) The name to be used when referring to the performance counter in C# code.

The value of this attribute must conform to the C# naming convention.

Help

(Required string.) The help text for the performance counter.
Type

(Required string.) The name of a System.Diagnostics.PerformanceCounterType enumeration value, which specifies the type of the performance counter.
