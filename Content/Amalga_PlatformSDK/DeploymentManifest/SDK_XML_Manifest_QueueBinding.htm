QueueBinding element

 

This topic describes the QueueBinding element.

Declares a queue binding for a service contract in a hosted .

Syntax

<Bindings>
   <QueueBinding
      [ RetryCount = "int" ]
      [ RetryDelay = "int" ]
      [ MaxPendingQueueSize = "int" ] />
</Bindings>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Bindings
Child elements None

Attributes

Attribute Description
RetryCount

(Optional integer.) The number of retries attempted to add a message to the queue. If this attribute is not specified, the default value is set to 5.
RetryDelay

(Optional integer.) The interval, in milliseconds, between attempts to add a message to the queue if a failure occurs. If this attribute is not specified, the default value is set to 30.
MaxPendingQueueSize

(Optional integer.) The maximum number of outstanding messages that can be maintained on the queue. If this attribute is not specified, the default value is set to 50.
