Resources element

 

This topic describes the Resources element.

Declares the list of resources exposed by the hosted application to users, computers, and hosted services.

Syntax

<ApplicationAuthorization>
   <Resources>
      <Resource>...</Resource>
   </Resources>
</ApplicationAuthorization>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 1-1: Required element that can occur once and only once.

Element relationships

Relationship Element
Parent elements ApplicationAuthorization
Child elements Resource

Remarks

The Resource elements contained by the Resources element declare what resources are available to other users, computers, and hosted services from the hosted application. If no Resource elements are specified, no application views or service operations are available to other users, computers, or hosted services.
