Edit an application manifest or service manifest

 

This topic describes how to use Manifest Editor to edit the elements and attributes of an application manifest or service manifest.

An application manifest provides the design-time and run-time metadata used to identify and configure a hosted application. Similarly, a service manifest provides the design-time and run-time metadata used to not just identify and configure a hosted or , but also to expose, as needed, the functionality of that hosted service.

Both application manifests and service manifests are XML files, and can be edited in using either the XML editor included with or the Manifest Editor included with .

The best practice is to use the Manifest Editor, to ensure well-formed XML and properly specified element and attribute values.

The following procedure describes how to use the Manifest Editor to edit an application manifest or service manifest in .

To edit an application manifest or service manifest with Manifest Editor

  1. In Solution Editor, right-click the application manifest or service manifest to be edited, and then select Edit Manifest.
  2. Edit the elements and attributes presented in the Manifest Editor as needed.
  3. Click OK.
