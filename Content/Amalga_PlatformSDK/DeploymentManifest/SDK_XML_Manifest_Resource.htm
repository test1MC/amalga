Resource element

 

This topic describes the Resource element.

Declares a specific resource, such as an application view from the virtual schema or a service operation from a hosted , exposed by the hosted application to users, computers, and hosted services.

Syntax

<Resources>
   <Resource
      Type = "AppView" | "Service"
      Name = "string"
      Operation = "ReadOnly" | "ReadWrite" | "*" | "string" />
</Resources>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-n: Optional element that can occur one or more times.

Element relationships

Relationship Element
Parent elements Resources
Child elements None

Attributes

Attribute Description
Type

(Required string.) The type of resource being exposed. The following values are allowed.

Value Description
AppView An from the virtual schema for the hosted application.
Service A service operation from a hosted for the hosted application.
Name

(Required string.) The name of the resource being exposed.
If AppView is specified for the Type attribute, then the value of this attribute must match the name of an application view specified in the data manifest for the hosted application.
If Service is specified for the Type attribute, then the value of this attribute must match the name of a hosted specified in a service manifest for the hosted application.

Operation

(Required string.) The name of the operation being exposed for the resource. The following values are allowed.

Value Description
ReadOnly Allowed only if AppView is specified in the Type attribute.
Allows read-only access to the specified in the Name attribute.
ReadWrite

Allowed only if AppView is specified in the Type attribute.

Allows read/write access to the specified in the Name attribute.
*

Allowed only if Service is specified in the Type attribute.

Allows access to all public operations exposed by the hosted specified in the Name attribute.
string

Allowed only if Service is specified in the Type attribute.

If string is set to the name of a public operation exposed by the hosted specified in the Name attribute, access is allowed to only that specified operation.

Remarks

The Resource element declares the availability of a given resource to other users, computers, and hosted services from the hosted application. If a Resource element is not specified for a given resource, the resource is unavailable to other users, computers, or hosted services.
