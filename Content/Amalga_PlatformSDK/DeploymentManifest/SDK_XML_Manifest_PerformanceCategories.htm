PerformanceCategories element

 

This topic describes the PerformanceCategories element.

Declares the list of performance categories and performance counters for the hosted or hosted .

Syntax

<DeploymentManifest>
   <PerformanceCategories>
      <PerformanceCategory>...</PerformanceCategory>
   </PerformanceCategories>
</DeploymentManifest>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality

0-1: Optional element that can occur once and only once.

This element can be included only in manifests and manifests. Other deployment manifest types do not use this element.

Element relationships

Relationship Element
Parent elements DeploymentManifest
Child elements PerformanceCategory

Remarks

The performance categories and performance counters included in this element are displayed in the Doors Performance Monitoring tool.
