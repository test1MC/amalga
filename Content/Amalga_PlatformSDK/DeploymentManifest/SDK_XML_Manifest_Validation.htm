Validation element

 

This topic describes the Validation element.

Provides a validation rule for the declared setting.

Syntax

<Setting>
   [ <Validation>string</Validation> ]
</Setting>

Element characteristics

Characteristic Description
Data type and length String
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Setting
Child elements None

Remarks

This element allows you to optionally specify a validation rule that can be evaluated when the setting is changed at run-time.
