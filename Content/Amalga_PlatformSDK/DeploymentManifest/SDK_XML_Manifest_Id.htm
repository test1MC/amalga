Id element

 

This topic describes the Id element.

Contains the GUID that identifies a hosted application.

Syntax

<Dependency>
   <Id>GUID</Id>
</Element>

Element characteristics

Characteristic Description
Data type and length String, representing a GUID.
Default value None
Cardinality 1-1: Required element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Dependency
Child elements None

Remarks

The value of this element is used to identify a hosted application referenced in a Dependency element. The value of this element must match the value for the Id attribute of the DependencyManifest element from the application manifest of the hosted application referenced in the Dependency element.
