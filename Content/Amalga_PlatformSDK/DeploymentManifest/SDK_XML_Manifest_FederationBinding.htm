FederationBinding element

 

This topic describes the FederationBinding element.

Declares a federation binding for a service contract in a hosted .

Syntax

<Bindings>
   <FederationBinding [ StreamedResponse = boolean ] />
</Bindings>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 0-1: Optional element that can occur once and only once.

Element relationships

Relationship Element
Parent elements Bindings
Child elements None

Attributes

Attribute Description
StreamedResponse

(Optional Boolean.) Specifies whether the response is streamed. If this attribute is not specified, then the default value is set to false.
