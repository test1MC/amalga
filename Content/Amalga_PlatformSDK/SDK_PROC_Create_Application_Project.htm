Create a new hosted application project

 

This topic describes how to create a new Amalga application project in .

All hosted applications in must contain an application manifest, and any hosted application that accesses data from also typically contains a data manifest. In , this metadata is contained within a Visual Studio project, created from a project template named Amalga Application Project. This template provides an application manifest, which contains the metadata for a hosted application.

A solution that contains hosted applications, including hosted services, must contain at least one project created from this template. Other projects, such as an Amalga WCF service project, must contain a reference to a project created from this template. Multiple Amalga application projects can be contained in the same solution.

When creating a new hosted application, the best practice is to first create a new solution that contains a project based from the Amalga Application Project template, and then add a data manifest or other projects to the new solution. The following procedure describes how to create a new solution containing such a project.

To create a new solution containing an Amalga application project

  2. From the File menu, click New, and then click Project.
  3. In the left pane, select Installed Templates, and then select Visual C#.
  4. In the middle pane, select Amalga Application Project.
  5. In the Name box, type a name for the new Amalga application project.
  6. In the Location dropdown, type or select a location in which to save the new solution and project, or click Browse to select a location.

  7. Select Create directory for solution if you want to create a new folder in the location specified in the Location dropdown, using the name specified in the Solution name box, for solution files.

  8. If Create directory for solution is selected, type a name for the new solution, if needed, in the Solution name box.

    If Create directory for solution is not selected, skip this step.

  9. Click OK.
