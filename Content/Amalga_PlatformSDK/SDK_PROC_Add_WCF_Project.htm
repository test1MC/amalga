Add a new hosted service project

 

This topic describes how to add a new or project to an existing solution containing an Amalga application project in .

To create a new hosted or hosted , you can use, respectively, the Amalga WCF Service Project or Amalga Doors Worker Project template. These templates provide a service manifest and sample classes for developing a hosted service. These templates also provide references to the namespaces most commonly used while developing a hosted service.

To include a hosted service in your hosted application, the solution for the hosted application must meet the following requirements:

  • The solution must contain an application project, created from the Amalga Application Project template.
  • The project for the hosted service must contain a reference to the Amalga application project.

The following procedure describes how to add a new project, based on the Amalga WCF Service Project or Amalga Doors Worker Project template, to an existing solution that already contains an application project based on the Amalga Application Project template.

To add a new project for a hosted service to an existing solution

  1. From the File menu, click New, and then click Project.
  2. In the Category pane, select Installed Templates, and then select Visual C#.
  3. In the Templates pane, select either Amalga WCF Service Project or Amalga Doors Worker Project, depending on whether you want to develop, respectively, a hosted or a hosted .

  4. In the Name box, type a name for the new project.

    Ensure that the name for the new project is unique within the solution. Otherwise, an error occurs.

  5. In the Location drop-down, type or select a location in which to save the new project, or click Browse to select a location.

  6. In the Solution drop-down list, select Add to solution.

  7. Click OK.
  8. In Solution Explorer, right-click the new project and then click Add Reference.
  9. Click the Projects tab, and then select the Amalga application project for the hosted application that will contain the hosted service.
  10. Click OK.
