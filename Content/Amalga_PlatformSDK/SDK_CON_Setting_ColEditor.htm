Setting Collection Editor

 

This topic describes the Setting Collection Editor, a tool integrated into that provides a user interface for editing configuration settings in application manifests and service manifests.

The Setting Collection Editor is a property editor that you can use within the integrated development environment (IDE) to edit the configuration settings contained in application manifests and service manifests.

You can display the Setting Collection Editor from within the Manifest Editor by selecting the Settings property, under the Data section, and then clicking the ellipsis button (...). The following graphic illustrates the Setting Collection Editor as it appears in .
