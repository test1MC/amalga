PerfCounterCategory Collection Editor

 

This topic describes the PerfCounterCategory Collection Editor, a tool integrated into that provides a user interface for editing performance counter categories in service manifests.

The PerfCounterCategory Collection Editor is a property editor that you can use within the integrated development environment (IDE) to edit the performance counter categories contained in service manifests.

You can display the PerfCounterCategory Collection Editor from within the Manifest Editor by selecting the PerfCounterCategories property, under the Data section, and then clicking the ellipsis button (...).
