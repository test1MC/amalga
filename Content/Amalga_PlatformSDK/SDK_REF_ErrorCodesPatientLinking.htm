Patient linking service event codes

 

This topic lists the event codes used by the patient linking service.

The PatientMatchLinkService service uses an enumeration to specify error codes when raising events. In , that enumeration is only defined within the Notification assemblies.

The following table lists the event codes used by the PatientMatchLinkService service.

Name Event ID Description
Unknown 12201 Unknown error.
InvalidArgument 12202 The service was passed an invalid argument.
AuthorizationFailure 12203 An authorization failure occurred.
RequestRecalled 12204 The request was recalled.
WorkflowClosed 12205 The workflow was closed.
InvalidOperation 12206 The operation could not be performed.
