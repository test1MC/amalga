Retrieving data with and

This topic describes how to use and Microsoft PowerPivot for to access data.

You can connect to Amalga data through Microsoft Excel 2010, using Microsoft PowerPivot for Microsoft Excel 2010, with the same security and access control as other Amalga applications. This topic describes the procedure for connecting to Excel.

Amalga allows connectivity with Microsoft Excel and PowerPivot so that you can access data from virtual schemas to create reports and analyze data. All data access will be subject to security and access control for the authenticated Amalga.

Use the following procedure to access data from Excel.

How To Access Amalga Data from Excel

  1. Open a new, blank worksheet in Excel.
  2. Click the Data tab.
  3. Click the From Other Data Sources dropdown and select From Data Connection Wizard.
  4. Choose Other/Advanced and click Next.

  5. In the Data Link Properties dialog box, make sure the Properties tab is selected. Select Microsoft OLE DB Provider for Amalga UIS from the list and click Next.

  6. In the Data Link Properties dialog box, make sure the Connectiontab is selected. Then:
    1. Enter your data source name. For example: http://<yourServerName>

    2. Type the information for the data location. For example: .\

    3. Enter login and password (if applicable). Check Allow saving password checkbox.
    4. For the Enter the initial catalog to use dropdown, choose the virtual schema you will use.
    5. Click the Test Connection button and verify that the connection works.
    6. Click OK.
  7. Select the application view to use and click Next.
  8. Select Save Password.

  9. Add any additional criteria and click Finish.

  10. Click Yes in the message about password security.

  11. In the Import Data dialog box, make selections for how you want to view data and where it should go, and then click OK.
