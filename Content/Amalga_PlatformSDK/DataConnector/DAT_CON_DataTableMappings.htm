DataTable and DataColumn mappings for AmalgaDbDataAdapter

 

This topic describes how the AmalgaDbDataAdapter object in supports the DataTableMapping object in ADO.NET.

The AmalgaDbDataAdapter object in , like the DataAdapter object in ADO.NET, uses a DataTableMappingCollection object from ADO.NET to contain zero or more DataTableMapping objects. The DataTableMapping object represents a mapped relationship between an application view and a DataTable object, and is used when the AmalgaDbDataAdapter populates a DataSet object. For more information about using the DataTableMapping object, see DataAdapter DataTable and DataColumn Mappings (ADO.NET), on Microsoft Developer Network at http://msdn.microsoft.com/library/ks92fwwh.aspx.
