DataConnector

 

This topic describes the DataConnector platform application, which interacts with the data providers included with to provide data access for other services and applications.

The DataConnector platform application provides data access services for platform features, hosted applications, and applications in . The DataConnectorService service, included with the DataConnector platform application, provides service contracts used by the following data providers:

  • .NET Framework Data Provider for

    The .NET Framework Data Provider for Amalga supports data access for applications using ADO.NET. For more information about the .NET Framework Data Provider for Amalga, see Retrieving and modifying data with on page 1.

  • OLE DB Data Provider for

    The OLE DB Data Provider for supports data access for older applications using OLE DB. The OLE DB Data Provider for uses the .NET Framework Data Provider for to handle communications with the DataConnector platform application.

The DataConnector platform application is not meant to be directly accessed by your application. Use the .NET Framework Data Provider for for data access within your application.
