Obtaining an AmalgaDbProviderFactory

 

This topic describes how to obtain an instance of the AmalgaDbProviderFactory class in .

The AmalgaDbProviderFactory class is the implementation of the DbProviderFactory class in ADO.NET, to support the factory design pattern used by ADO.NET for provider-independent data access.

The ADO.NET methodology for retrieving provider information, listing the installed provider factory classes, and creating an instance of the DbProviderFactory class can be used with . For more information about working with the DbProviderFactory class in ADO.NET, see Obtaining a DbProviderFactory (ADO.NET), on Microsoft Developer Network at http://msdn.microsoft.com/library/dd0w4a2z.aspx.

The AmalgaDbProviderFactory object does not implement the CreateDataSourceEnumerator method or the CreatePermission method from the DbProviderFactory class. does not implement the DbDataSourceEnumerator or CodeAccessPermission classes.

Example

The following example demonstrates how to use the AmalgaDbProviderFactory class to instantiate an AmalgaDbConnection object, cast as a DbConnection object.

public DbConnection CreateDbConnection(
    string providerName)
{
    // Define a variable for the DbConnection object.
    DbConnection connection = null;

    // Instantiate the AmalgaDbProviderFactory class, and
    // then use it to instantiate an AmalgaDbConnection class and
    // cast it as a DbConnection object.
    // For hosted applications, you can use the static function
    // GetVirtualSchemaConnectionString() to obtain a connection
    // string.
    try
    {
        AmalgaDbProviderFactory factory = new AmalgaDbProviderFactory();
        connection = factory.CreateConnection();
        connection.ConnectionString = GetVirtualSchemaConnectionString();
    }
    catch (Exception ex)
    {
        // Set the connection to null if a DbConnection instance was created.
        if (connection != null)
        {
            connection = null;
        }
        Console.WriteLine(ex.Message);
    }

    // Return the connection.
    return connection;
}
