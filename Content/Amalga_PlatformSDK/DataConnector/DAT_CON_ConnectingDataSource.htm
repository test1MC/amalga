Connecting to a data source

 

This topic describes how to connect to a DataConnector instance using .

In , you use an AmalgaDbConnection object to connect to a specific instance of the DataConnector platform application by supplying the necessary authentication information in a connection string. The AmalgaDbConnection object implements the DbConnection class from ADO.NET.
