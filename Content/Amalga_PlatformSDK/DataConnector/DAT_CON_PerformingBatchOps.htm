Performing batch operations with AmalgaDbDataAdapter

 

This topic describes how to use the AmalgaDbDataAdapter object, in conjunction with the AmalgaDbCommandSet object, to perform batch operations in .

The AmalgaDbDataAdapter provides batch support, to allow update operations from a DataSet or DataTable object to be grouped and sent to an instance of the DataConnector platform application, instead of sending one update operation at a time for each updated row. The AmalgaDbDataAdapter object implements the UpdateBatchSize property from the DbDataAdapter object in ADO.NET. For more information about performing batch operations, see Performing Batch Operations Using DataAdapters (ADO.NET), on Microsoft Developer Network at http://msdn.microsoft.com/library/aadf8fk2.aspx.
