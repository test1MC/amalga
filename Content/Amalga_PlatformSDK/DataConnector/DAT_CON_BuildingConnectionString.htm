Building a connection string

 

This topic describes how to build a connection string for a DataConnector instance in .

In , a connection string is used by an AmalgaDbConnection object to connect to an instance of the DataConnector platform application. For more information about connection strings in ADO.NET, see Connection Strings, on Microsoft Developer Network at http://msdn.microsoft.com/library/ms254978.aspx.

does not include a provider-specific implementation of the DbConnectionStringBuilder class from ADO.NET. However, you can use the GetVirtualSchemaConnectionString method, included in the AmalgaService class and available to hosted and instances, to provide an appropriate connection string for the hosted application.

The following example connection string illustrates a typical connection string for a hosted application:

"Data Source=https://computer.domain.com:8889;Initial Catalog=HostedApplication.TestInstance;Integrated Security=True"

Connection string properties

The AmalgaDbConnection object recognizes the following connection string properties:

Property Description
User ID

Optional string. The user ID to be used when connecting to the instance.

The User ID connection string property is not supported.
Password

Optional string. The password to be used when connecting to the instance.

The Password connection string property is not supported.
Connection TimeOut

Optional integer. The time, in seconds,

The default value for this property is set to 30.
Connect TimeOut See Connection TimeOut.-, above.
Data Source Required string. The fully qualified domain name (FQDN) of the physical host for the Topology platform feature.
Integrated Security

Required Boolean. If set to True, Yes, or SSPI, indicates that the current Doors account credentials are used for authentication. Otherwise, the values provided in the User ID and Password connection string properties are used for authentication.

The default value for this property is set to True.

The User ID and Password connection string properties are not supported.

AllowInputPrompt  
AuthToken Optional string. This connection string property is deprecated.
Initial Catalog Required string. The full name of the application instance.
Persist Security Info Optional Boolean. Indicates whether security-sensitive information, such as the password, is returned as part of the connection if the connection is open or has ever been in an open state.
