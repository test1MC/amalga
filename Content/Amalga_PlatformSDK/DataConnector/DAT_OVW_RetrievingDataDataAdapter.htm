Retrieving data using AmalgaDbDataAdapter

 

This topic describes how to use the AmalgaDbDataAdapter class in to retrieve data.

The AmalgaDbDataAdapter class in provides an implementation of the DbDataAdapter class in ADO.NET. For more information about the DbDataAdapter class in ADO.NET, see DbDataAdapter Class, on Microsoft Developer Network at http://msdn.microsoft.com/library/system.data.common.dbdataadapter.aspx. The AmalgaDbDataAdapter class not only provides interaction between the DataConnector platform application in and a DataSet or DataTable object in ADO.NET, it also provides batch operation functionality in conjunction with the AmalgaDbCommandSet object.
