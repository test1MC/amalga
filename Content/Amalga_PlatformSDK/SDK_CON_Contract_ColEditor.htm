Contract Collection Editor

 

This topic describes the Contract Collection Editor, a tool integrated into that provides a user interface for editing service contracts in service manifests.

The Contract Collection Editor is a property editor that you can use within the integrated develMyNewTopicopment environment (IDE) to edit the service contracts contained in service manifests.

You can display the Contract Collection Editor from within the Manifest Editor by selecting the Contracts property, under the Data section, and then clicking the ellipsis button (...). The following graphic illustrates the Contract Collection Editor as it appears in .
