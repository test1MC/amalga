Scenario and event markers for platform services and platform applications

 

The following tables describe the scenario markers and event markers provided for the platform features and platform applications included with . The globally unique identifier (GUID) is used in the performance log to identify a performance log entry for the start and stop of a scenario or event. For more information about the performance log, see Performance log on page 1.
