Data modeling

 

This topic provides an overview of data modeling, including the basic tasks involved with creating and defining a data manifest in .

Modeling a virtual schema for a hosted application in is performed by using to create a data manifest, define application views, and map those application views to underlying data structures. Once the data model is complete, then preparing the data manifest for deployment is needed to ensure that the publishing and deployment tools can use the data manifest to generate the appropriate metadata and data structures.
