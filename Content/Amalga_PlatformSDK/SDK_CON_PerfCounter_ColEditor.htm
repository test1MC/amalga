PerfCounter Collection Editor

 

This topic describes the PerfCounter Collection Editor, a tool integrated into that provides a user interface for editing performance counters in service manifests.

The PerfCounter Collection Editor is a property editor that you can use within the integrated development environment (IDE) to edit the performance counters contained in service manifests.

You can display the PerfCounter Collection Editor from within the PerfCounterCategory Collection Editor by selecting the PerfCounters property, under the Design section, and then clicking the ellipsis button (...).
