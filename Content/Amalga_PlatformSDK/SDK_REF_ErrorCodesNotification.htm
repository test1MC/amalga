Notification service event codes

 

Lists the event codes used by the Notification service.

The NotificationService service uses an enumeration to specify error codes when raising events. In , that enumeration is only defined within the Notification assemblies.

The following table lists the event codes used by the Notification service.

Name Event ID Description
UnknownError 12701 Unknown error
NotImplemented 12702 Not implemented
NullArgument 12703 Argument is null
InvalidArgument 12704 Invalid argument
NotificationDefinitionAlreadyExists 12705 Notification type definition already exists
NotificationDefinitionMissing 12706 Notification type definition is not available
InvalidNumberOfArguments 12707 Argument number needs to be >= 0
WrongNumberOfArguments 12708 Wrong number of argValues for the notification type
InvalidRequestId 12709 Invalid RequestId for notification
InvalidSeverity 12710 DisplaySeverity is invalid
InvalidNotificationTypeId 12711 Attempt to reference an unavailable type ID, or save using a type ID that is already in use
NotificationDefinitionAlreadyDeleted 12712 Attempt to delete a type definition that has already been deleted
InvalidResourceIdSyntax 12713 Notification references an invalid resource ID
InvalidQualifiedTypeNameSyntax 12714 The QualifiedNameAndVersion value of the type definition is invalid
InvalidStateForTheOperation 12715 Operation is undefined for the current notification state
