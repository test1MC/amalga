Comparing data manifests

 

This topic describes how to compare data manifests using the VirtualSchema platform feature in .

You can invoke the CheckIfDataManifestsAreSame operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking CheckIfDataManifestsAreSame, you must specify two strings representing the XML contents of the two data manifests to be compared.

CheckIfDataManifestsAreSame reconstructs the data manifests and performs an object comparison between them, to avoid issues related to string comparison (such as the ordering of entity definitions within the data manifest.)
