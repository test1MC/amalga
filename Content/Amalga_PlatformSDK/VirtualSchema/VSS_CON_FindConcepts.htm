Find concepts assigned to entities

 

This topic describes how to find concepts for entities in .

You can find concepts assigned to an array of specified entities by invoking the FindConcepts operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking FindConcepts, you must specify a ConceptInfoQuery object that represents the specified entities for which assigned concepts are to be retrieved. For more information about concepts, see Concepts on page 1.

Working with the ConceptInfoQuery object

The ConceptInfoQuery object represents the set of specified entities for which concepts are to be found. The ConceptInfoQuery object has the following properties.

Property name Description
Entities

An array of EntityInfo objects representing the entities to be examined.

You must specify at least one EntityInfo object that contains information about an entity in the ; otherwise, an error occurs when invoking the operation.

Invoking the operation

When the operation is invoked, the VirtualSchemaService service retrieves the identifiers and names of every entity and entity property in the , mapping identifiers to fully qualified names for entities and entity properties. Once this is complete, the service then retrieves all of the concepts mapped to the entities and entity properties, returning a Dictionary object containing a ConceptTags object for each entity and entity property.

Working with the ConceptTags object

The ConceptTags object contains an array of ConceptInfo objects for a given entity or entity property, in which each ConceptInfo object represents a concept associated with that entity or entity property. The ConceptTags object has the following properties.

Property name Description
ObjectQName A string representing the fully qualified name of the entity or entity property.
Tags An array of ConceptInfo objects representing the concepts associated with the entity or entity property identified in the ObjectQName property.

Working with the ConceptInfo object

The ConceptInfo object represents a concept, stored and managed by the Terminology platform application. The ConceptInfo object has the following properties

Property name Description
ConceptCode A string representing the concept code for the concept.
ConceptId A string representing the unique identifier for the concept.
ConceptMeaning A string representing the meaning for the concept.
Terminology A string representing the terminology that contains the concept.
