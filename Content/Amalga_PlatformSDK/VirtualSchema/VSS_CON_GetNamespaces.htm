Getting entity namespaces

 

This topic describes how to get entity namespaces using the VirtualSchema platform feature in .

You can get the namespaces of entities published in the by invoking the GetNamespaces operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service.

When invoked, the GetNamespaces operation returns an ordered collection of strings representing all of the distinct namespaces for entities published in the . The value of each string is equivalent to the value of the Schema property for each Entity or EntityContainer object included in the .
