Validating a defining query

 

This topic describes how to validate a defining query using the VirtualSchema platform feature in .

You can validate a defining query, such as the SQL statement in the DefiningQuery property of an EntityBinding object, by invoking the ParseDefiningQuery operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking ParseDefiningQuery, you must specify a string that contains a valid SQL statement.

If the SQL statement is valid, the operation returns a DefiningQueryInfo object that contains the validated query text of the defining query. Otherwise, an exception is thrown containing any errors identified in the SQL statement.
