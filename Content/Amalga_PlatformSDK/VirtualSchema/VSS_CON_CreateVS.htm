Creating a virtual schema

 

This topic describes how to create a virtual schema using the VirtualSchema platform feature in .

You can create a virtual schema by invoking the CreateVirtualSchema operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking CreateVirtualSchema, you must specify the name of the application instance and the contents of the data manfiest which represents the virtual schema. For more information about data manifests, see Data manifests on page 1.
