Creating an entity association

 

This topic describes how to create entity associations using the VirtualSchema platform feature in . .

Entity associations define the associations between application views in the conceptual schema, as well as between entities in the storage schema, for a virtual schema. You must first use an EntityAssociation object to define the entity association to be created, and then invoke the CreateEntityAssociation operation of the IVirtualSchemaManagementService service contract to create that entity association in the virtual schema.

Working with the EntityAssociation object

The EntityAssociation object represents an entity association in a virtual schema. The EntityAssociation object has the following properties.

Property name Description
EntityContainer An EntityContainer object representing the entity container that holds the entity association.
ForeignEntityConstraint An EntityConstraint object representing a foreign key constraint to another entity.
ForeignEntityMultiplicity A string representing the multiplicity of the foreign key constraint in ForeignEntityConstraint.
Id A System.Guid object representing the globally unique identifier (GUID) of the EntityAssociation object.
UniqueEntityConstraint An EntityConstraint object representing a unique or primary key constraint.
UniqueEntityMulitplicity A string representing the multiplicity of the unique or primary key constraint.

Working with the EntityConstraint object

If the entity association represents a foreign key constraint, you must define a corresponding EntityConstraint object that describes the foreign key constraint as the value of the ForeignEntityConstraint property for the EntityAssociation object. Similarly, if the entity association represents a unique constraint or primary key constraint, you must define a corresponding EntityConstraint object that describes the unique or primary key constraint as the value of the UniqueEntityConstraint property for the EntityAssociation object.

The EntityConstraint object has the following properties.

Property name Description
ConstraintType An integer that describes the type of constraint.
Entity An Entity object representing the entity to which the constraint applies.
EntityConstraintProperties An array of EntityConstraintProperty objects representing user-defined properties for the constraint.
Id A System.Guid object representing the globally unique identifier for the constraint.
Name A string containing the name of the constraint.
