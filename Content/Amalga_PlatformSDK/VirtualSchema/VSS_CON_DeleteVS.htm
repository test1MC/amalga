Deleting a virtual schema

 

This topic describes how to delete a virtual schema using the VirtualSchema platform service in .

You can delete an entity by invoking the DeleteVirtualSchema operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking DeleteVirtualSchema, you must specify the name of the virtual schema instance to be deleted.
