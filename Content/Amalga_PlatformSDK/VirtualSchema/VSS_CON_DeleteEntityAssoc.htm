Deleting an entity association

 

This topic describes how to delete an entity association using the VirtualSchema platform feature in .

You can delete an entity association by invoking the DeleteEntityAssociation operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking DeleteEntityAssociation, you must specify an EntityAssociationInfo object that represents the entity association to be deleted.

Working with an EntityAssociationInfo object

The EntityAssociationInfo object has the following properties.

Property name Description
ExtensionData An ExtensionDataObject object representing extra data used when serializing and deserializing the object.
Id A System.Guid object representing the globally unique identifier (GUID) of the entity association.
