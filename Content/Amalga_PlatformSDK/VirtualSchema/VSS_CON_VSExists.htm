Confirm the existence of a virtual schema

 

This topic describes how to confirm the existence of a virtual schema using the VirtualSchema platform service in .

You can confirm the existence of a virtual schema instance by invoking the VirtualSchemaInstanceExists operation, from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking VirtualSchemaInstanceExists, you must provide the instance name of the virtual schema to be confirmed.

When invoked, the operation returns a Boolean value indicating whether the specified virtual schema instance exists.
