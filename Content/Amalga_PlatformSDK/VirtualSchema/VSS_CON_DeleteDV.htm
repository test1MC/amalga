Deleting a dynamic view

 

This topic describes how to delete a dynamic view using the VirtualSchema platform feature in .

You can delete a dynamic view by invoking the DeleteDynamicView operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking DeleteDynamicView, you must specify the name of the dynamic view, the name of the virtual schema that contains the dynamic view, and the name of the hosted application that created the dynamic view.
