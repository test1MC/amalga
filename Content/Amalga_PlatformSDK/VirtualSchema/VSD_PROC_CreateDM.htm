Create a data manifest

This topic describes how to create a data manifest for an Amalga application project in .

The following procedure describes how to create a data manifest for an existing Amalga application project in .

Only one data manifest can be added to the Amalga application project for a hosted application. If you add more than one data manifest, an error occurs when the hosted application is published or deployed.

To add a new data manifest in

  1. In Solution Explorer, select the Amalga application project for which the data manifest is to be created.

    A data manifest should be added only to a project created from the Amalga Application Project template.

  2. On the Project menu, click Add New Item.
  3. In the Category pane, select Installed Templates, and then select Visual C# Items.
  4. In the Templates pane, select Amalga Data Manifest.
  5. Click Add.
