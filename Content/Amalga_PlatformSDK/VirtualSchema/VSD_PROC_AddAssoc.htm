Add an association

 

This topic describes how to add a scalar property to an application view using in for .

You can add an association to a read-only or read-write application view by using in . For more information about associations, see Entity associations on page 1.

To add an association to an application view

  1. In the visual design surface of , right-click the application view, click Add, and then click Association.

  2. In the Add Association dialog box, specify the name of the association in Association Name.
  3. In the leftmost End frame, ensure that Entity is set to the name of the application view right-clicked in step 1.
  4. Select the desired multiplicity for the application view in Multiplicity.
  5. Ensure that Navigation Property is selected, and that the appropriate name of the navigation property to be created in the application view is specified.

  6. In the rightmost End frame, select the other application view to be referenced by the association in Entity.

    You can define a self-association by selecting the same entity for both ends.

  7. Select the desired multiplicity for the other application view in Multiplicity.
  8. Ensure that Navigation Property is selected, and that the appropriate name of the navigation property to be created in the other application view is specified.
  9. If applicable, ensure that Add foreign key properties to the <entity name> Entity is selected.
  10. Click OK.
