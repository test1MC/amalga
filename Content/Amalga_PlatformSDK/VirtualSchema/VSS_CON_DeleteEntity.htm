Deleting an entity

 

This topic describes how to delete an entity from

You can delete an entity by invoking the DeleteEntity operation from the IVirtualSchemaManagementService service contract for the VirtualSchemaService service. When invoking DeleteEntity, you must specify the name of the namespace that contains the entity and the name of the entity to be deleted.

An entity cannot be deleted if any of the following conditions are true:

  • The entity has already been bound to an instance of a virtual schema.
  • The entity has one or more Access Control Framework (ACF) rules associated with it.
