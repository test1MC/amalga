Finding a DCN event type

 

This topic describes how to find an existing DCN event type by using the FindEvents and FindEventTypeForId operations.

 

 

 

asdf sadf

Before a hosted application can perform an operation involving an existing Data Change Notification (DCN) event type, such as activating or subscribing to a DCN event type, the existing DCN event type must first be identified and retrieved.

A hosted application can use two operations, FindEvents and FindEventTypeForId, in the IEventManagementService contract for the EventingManagerService service, to retrieve EventType objects representing DCN event types.

The FindEvents operation retrieves an array of EventType objects that represent all of the DCN event types created by the user account calling the FindEvents operation.

The FindEventTypeForId operation retrieves an EventType object for the DCN event type that matches the specified event identifier, if the DCN event type is owned by the same user account that called the FindEventTypeForId operation.
