Subscribing to a DCN event type

 

This topic describes how to subscribe to an existing DCN event type by using the Subscribe operation.

Hosted applications can subscribe to an existing DCN event type by defining a DCN event subscription with an EventSubscription object, specifying the event identifier for the existing DCN event type in the EventId property of the EventSubscription object, and then specifying the EventSubscription object while calling the Subscribe operation in the IEventManagementService contract for the EventingManagerService service.

The Subscribe operation performs the following actions when creating a DCN event subscription for an existing DCN event type.

  • Confirms that the DCN event type with the event identifier specified in the EventId property of the EventSubscription object exists.

  • Ensures that the user account that calls the Subscribe operation must match the value of the Owner property for the EventType object that represents the existing DCN event type to be subscribed.

    Only the user account that created the DCN event type can subscribe to a DCN event type. Otherwise, an error occurs.

  • Confirms that the user account calling the Subscribe operation is authorized to access the eventable resources associated with the specified existing DCN event type. Otherwise, an error occurs.
  • Creates a new DCN event subscription for the specified existing DCN event type.
