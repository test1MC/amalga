Manifest Editor

 

This topic describes the Manifest Editor, a tool integrated into that provides a user interface for editing application manifests and service manifests.

The Manifest Editor is a property editor that you can use within the integrated development environment (IDE) to edit the elements and attributes contained in application manifests and service manifests.
