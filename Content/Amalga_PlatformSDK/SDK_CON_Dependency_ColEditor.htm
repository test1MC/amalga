Dependency Collection Editor

 

This topc describes the Dependency Collection Editor, a tool integrated into that provides a user interface for editing application dependencies in application manifests and service manifests.

The Dependency Collection Editor is a property editor that you can use within the integrated development environment (IDE) to edit the application dependencies contained in application manifests and service manifests.

You can display the Dependency Collection Editor from within the Manifest Editor by selecting the Dependencies property, under the Data section, and then clicking the ellipsis button (...).
