Interacting with platform features

 

This topic describes how to interact with platform features.

platform features are integral components of the Amalga platform. Platform features, such as the Logger feature or Topology feature, provide functionality common to all Amalga installations. Each Amalga platform feature provides at least one service with which a hosted application can interact, to incorporate common Amalga functionality.
