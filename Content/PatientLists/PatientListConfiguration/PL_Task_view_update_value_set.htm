View or update a value set

 

This topic describes how to use the ValueSetUpdateScript to view or update a value set.

After a value set is created, you can view its properties and update any of the properties using the ValueSetUpdateScript.sql script located in the Samples\DataExplorer folder of the installation media.

To view a value set

  1. Provide a value for either the @ValueSetID or @ValueSetName or both. Leave all other variables as empty strings.

    You can only view one value set at a time.

  2. Run the script to retrieve and display data associated with the value set.

To modify a value set

  1. Provide a value for either the @ValueSetID or @ValueSetName, or both.

    You can only update one value set at a time.

    If both a name and ID pattern are supplied, the value set must match both for the update to occur.

  1. In @UpdateValueSet, enter any single character; for example, 1.
  2. Enter values for any variables that you want to change, including the query itself.

  3. Run the script.

    Basic information about the value set and properties modified will be displayed.

To remove a value set

  • Use the ValueSetUpdateScript to remove all entity attribute bindings.

    All lists using the value set will continue to work.
