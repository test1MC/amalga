Create custom criteria

 

This topic describes how to create a criterion or criteria that will be used to build queries based on the data model.

Custom criteria can be based on the attributes of any entity with a foreign key to the Patient entity. All data types supported by the entity model are supported for use as custom criteria except for the Time data type.

To create a custom criterion

  1. Create a definition in the tbl_SchemaConfiguration table. The EntityName and AttributeName columns must correspond to an entity attribute that is part of the application view.
  2. Provide a label in the tbl_SchemaConfigurationDisplayName table.

  3. For a new entity, in the tbl_LinkedEntity table, specify how the entity is related to the Patient entity.

    Changes will take effect once the cache is updated on the server.

To hide a custom criterion

  • In the tbl_SchemaConfiguration table, change the IsVisibleQueryBuilder column to 0.
