scenarios

 

This topic describes typical scenarios for creating, viewing, editing, and sharing patient lists.

To get started with patient lists, use the following procedures.

Create a patient list

  1. Create a patient list. See Create a patient list on page 1.
  2. Edit columns. See Edit a list on page 1.
  3. Share the list. See Share a list on page 1.

View a list

  1. Browse the patient lists, organize the lists, and select a patient list to view. See Manage on page 1.
  2. View reports associated with a list. See View the pane.
  3. View the chart for a patient on the list. See Start .

Edit a list

  1. Edit the list properties. See Edit a list on page 1.
  2. Edit sharing permissions. See Share a list on page 1.
  3. Edit columns. See Edit columns on page 1.
  4. Edit list criteria. See Edit list criteria on page 1.

Share a list

  1. Browse the patient lists, organize the lists, and select a patient list to view. See Organize on page 1.
  2. Edit sharing permissions. See Share a patient list on page 1.
