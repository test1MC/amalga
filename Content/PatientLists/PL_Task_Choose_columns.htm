Edit patient list columns

 

This topic describes how to edit the columns that will be displayed in a patient list.

Once a patient list is created, you can select the columns that will be displayed in the patient list. You can also reorder the columns, and remove columns if you have a minimum of edit permissions.

You cannot remove required columns.

If you have view-only permissions for the list, you will not see the Edit list button.

To edit columns

  1. From the patient list, click Edit list.
  2. Click Edit columns.

  3. Expand a category and then click the column that you want to add.

    When you select an encounter-based column for the first time, a message appears informing you that the patient may appear more than once in the list if there are multiple encounters.

    When you add a new column, all grouping or sorting will be removed.

To reorder columns

  • Grab the handle for the column and drag it to the desired location.

To remove a column

  • Click Delete () in the column you want to delete.

    You must have at least one column in the grid.
