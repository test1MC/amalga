View the pane

 

This topic explains how to view a report or reports associated with a patient list.

If a patient list has associated graphs, charts, or reports, you can display them from the page.

To view the pane

  1. From the portal, click .
  2. Click the patient list you want to display.
  3. Click the Open the pane icon ( ).

  4. The reports associated with this list will be displayed, along with the patients in the list.

    If there are no reports associated with this list, a message will be displayed, and you can click Create a report if you have report author privileges. For more information, see Create a report.
