roles and permissions

 

This topic describes the permission levels related to sharing and what options you will see when you view the lists.

Depending on your access level, some options in will be unavailable. The table below indicates which functions are available and editable.

             
UI Element Functions   Full control View, edit & delete View & edit View-only
Edit list     Available Available Available N/A
  Edit list criteria   Available Available Available N/A
  Edit columns   Available Available Available N/A
  Edit list properties   Available Available Available N/A
    Basic Info tab Editable Editable Editable Viewable, but not editable
    Sharing tab Editable Viewable, but not editable Viewable, but not editable Viewable, but not editable
More     Available Available Available Available
  View list properties   Available Available Available Available
  Delete list   Available Available N/A N/A
