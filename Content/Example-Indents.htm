Example-Indents Edit

All of these indentations can be accomplished without using styles that force an indent.

Reasons to indent: 

  • Align unnumbered/unbulleted paragraph between list items
  • Align lists under a glossary definition or other outside-list, already-indented paragraph
  • Multiple single item lists without bullets or numbers

Non-list Paragraphs between List Items

In this example, unnumbered/unbulleted paragraphs are aligned within a list. This alignment requires making sure that the paragraphs are within the OL or UL tag for the list. No other special tagging is required to achieve the indentation in the output. Remember to check the preview window to see how the content will line up.

Toadd an non-list paragraph to a list item

  1. This is a standard list item.

  2. This is in a p block because it has a following non-list paragraph. You get a structure like this by typing in this line, right-clicking on the <li> block and selecting Make paragraph item(s). Then you go to the end of this text, and press ENTER to get the next paragraph, which will be without a number.

    This paragraph is also in a "p" (no class) block and it will line up under the text in step 2. You would do a note, tip, etc. the same way. Click the preview button to see what it looks like.

    To make the next numbered item, press ENTER, then click on the yellow and black arrow icon.

  3. Now you have another numbered item.

This paragraph is tagged "p" but is outside the OL tag so it won't align with the list.

For information on lining up "p" classes, such as notes, tips, cautions, and warnings, within lists, see the "Example-Note-Tip-Caution-Warning.htm" topic in the Content root.
