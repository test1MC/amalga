View and interact with reports

 

This topic explains the tasks that can be performed by all users in the pane.

In the pane, you can view any reports associated with a patient list and you can perform a variety of report-related tasks.

Tabular

Tabular reports present data in a grid or table rather than as a chart or graph.

In addition to the functions listed above, the following additional functions can be performed on tabular reports.

To view a different page

  • Do one of the following:
  • Type a page number in the Current Page box, and then press Enter.
  • Click the right-facing single arrow to advance one page
  • Click the left-facing single arrow to go back one page.
  • Click the First Page icon () to go to the first page.
  • Click the Last Page icon ()to advance to the last page.

To zoom the report

  • In the Zoom drop-down list box, click a zoom option.

To export the report

  • In the Export drop-down list box, click one of these options:
  • Excel
  • PDF
  • Word
