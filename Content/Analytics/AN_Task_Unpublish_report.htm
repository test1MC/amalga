Unpublish a report

 

This topic describes how to unpublish a report.

If you have previously published a report to make it visible to all users with access to the associated list, you can "unpublish" it so that it is invisible to users with view-only access. You might also unpublish a report that you want to delete. An unpublished report can later be previewed and republished to make it visible to all users.

You must unpublish a report before you can delete it.
