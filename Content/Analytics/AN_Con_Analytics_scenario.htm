scenario

 

This topic describes a typical scenario for using .

  1. A report author creates a report using and saves the report. See Create a report.

  2. The report author imports, previews, and publishes the report. See Import a report.

    All users with at least view access to the patient list can now view the report.

  3. A clinician goes to the page and picks a patient list to view. See Manage on page 1.
  4. The clinician displays the associated reports. See View the pane.
  5. The clinician performs a variety of tasks on the reports. See Manage reports on page 1.
  6. The clinician filters the list to show only patients of interest. See Filter a list using a graph.
