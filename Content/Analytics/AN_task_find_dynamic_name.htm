Locate the name of a dynamic view

 

This topic describes how to locate the name of the dynamic view to use when accessing data from a patient list using an .

Dynamic views are automatically created each time a patient list is created or modified. You can use them to access patient list data using a

To locate the dynamic view name

  1. Open the report file.
  2. Expand Datasets.
  3. Right-click the dataset name and click Dataset Properties.

  4. The name of the dynamic view is located in the select statement.
