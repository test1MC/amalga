Tasks that report authors can perform

 

This topic lists the tasks that report authors can perform.

A user or user group who is granted specific permissions can act in the Report Author role. These permissions are granted via a command. For more information about setting report author user rights, see Assign a user role on page 1. The user rights are activated on a specific list when the user has a minimum of view and edit permissions for the list.

Only report authors can see the link to Create a report from a patient list in the pane.

Report authors can perform all the tasks that other users can perform, but in addition, they can perform the following tasks.
