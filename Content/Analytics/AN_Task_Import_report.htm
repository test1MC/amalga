Import a report

 

This topic describes how to import a report.

Once you have created a report, you need to import it into .

To import a report

  1. Click Import file.

  2. In the Import the file dialog box, click Browse, and then navigate to the location of your report.

    Ensure that the report you are importing was created from the patient list you are viewing.

  3. Click the report name, and then click Open.
  4. Fill in the report properties, and then click OK.
  5. Do one of the following:
  1. Click Preview, and then click Publish to publish the report.
  2. Click Preview, and then click Close to delay publishing the report. It will be displayed as an unpublished report; you can preview it and publish it later.

    You must preview the report before you can publish it. If there are problems, correct the issues and preview again.

    The report information is displayed in the list.
