Tasks that list editors can perform

 

This topic lists the tasks that users designated as list editors can perform.

List editors can perform all the tasks that users with view-only permissions can perform, but in addition, they can perform the following tasks. List editors will have a minimum of view & edit list permissions, but no report author privileges.
