Create a report from a patient list

 

This topic describes how to create a report from a patient list.

To create a report

  1. Click Create a report.

  2. Select the Don’t show me this message again check box to start later without seeing the message.

    If you previously selected that option, you will have the option to open or save a new report (.rdl) file. An empty placeholder will be created in the pane with Import and Cancel options.

  3. You will see a file dialog box in your browser.
  4. Click Open.

  5. Once the empty report is opened in , click the Office icon, click Save As and save the report to a known location on your local computer.

    By default, the report is opened in a temporary folder and may be deleted by the browser.

  6. Create the report.

    Do not modify the data source and data set (the columns in the associated list). Any modifications may result in an inability to import the report.

  7. Once you have created the report and saved it, return to the page and Import a report.
