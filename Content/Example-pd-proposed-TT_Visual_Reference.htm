[Example:] dialog box

 

This topic includes an image of the dialog box and describes its key elements.

Refine access for a given rule. For example, create SQL conditions to filter access to visits admitted in a particular facility, look up data from other entities, or map the current session user to its relationship to a patient record.

Access:

To start , see xref.

To access the dialog box from the : See xref. Authorize a role for access to an entity by adding or editing a rule; in the Add Custom Condition page of the dialog box, select the check box for where custom condition is true, and then click custom condition..

Ref# Name Description and tasks
1 Insert Selected Inserts the entity property that is selected under Available Attributes into the Custom SQL Filter window.
2 Custom SQL Filter Window for user-entered filter.
3 Available Attributes Lists all properties of the selected entity. Double-click a property to copy its name to the Custom SQL Filter window.
