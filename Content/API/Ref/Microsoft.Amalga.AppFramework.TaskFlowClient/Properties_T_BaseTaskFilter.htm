PagingVersionId
If set, indicates a paging query relative to the indicated task version. When set, indicates the returned tasks are older or newer than the indicated task version, depending on the Sorting properties. Paging requires that a MaxTaskNumber is set.
Scope
Indicates a scope of a query. See TaskScope enum description for details. Obligatory.
Sorting
Indicates result sorting for a query. See TaskSorting enum description for details. Obligatory.
TaskId
If set, indicates a task id to a query; all task updated from a task in a task flow share the same TaskId. When set, it needs to be different from empty Guid and indicate a task id returned from the service.
TaskState
If set, indicates a task state to be queried for. Amalga predefines some common state strings. When set, needs to be different from an empty string.
TaskType
If set, indicates a task type to be queried for. When set, needs to be a non empty string indicating a namespace.
UpdatorId
If set, indicates an UpdatorId string to be queried for.
VersionId
If set, indicates a specific version of a task to a query. When set, it needs to indicate a version returned from the service.
XPathFilter
If set, indicates an XPath string to be used on the xml in the Payload. When used, it needs to be a non empty string that is a valid XPath expression. Using XPath requires setting TaskType, a date range and MaxTaskNumber.
