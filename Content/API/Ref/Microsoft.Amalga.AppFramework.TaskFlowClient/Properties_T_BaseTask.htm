BaseTask Properties
BaseTask Class  See Also 

The BaseTask type exposes the following members.

Properties

  NameDescription
AssignedTo
A helper string indicating the agent assigned the task. Managed by the application.
CreatorId
Internal Amalga Id of the creator of the task. Managed by the service.
IsCurrent
Indicates that a task is the current task. For each task there is only one task version that is marked current. Managed by the service.
ModifiedBy
A helper string indicating the agent modifying the task. Managed by the application.
Options
If supplied, a set of flags related to the task version. Managed by the application.
Payload
String defining the task payload information. Must be a valid Xml string. Managed by the application. The service treats xml strings as xml fragments.
State
Indicates a state of the task version. State is an application defined string. Amalga predefines some common state strings in the task service client utility. Managed by the application.
TaskId
Task id identifying the task, that is a task workflow thread. Common to the task updated from a task. Managed by the service.
TaskType
Task type of the task. TaskType string has to be in a namespace, indicated by a dot separator. Managed by the application.
TimeCreated
Task version create time. Managed by the service.
UpdatorId
Internal Amalga Id of the updator of the task version. UpdatorId is the same as CreatorId for the initial task version. Managed by the service.
VersionId
Task version identifying an individual version of a task in a task workflow thread. Managed by the service.
