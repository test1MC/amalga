Members

Member nameDescription
Default
Write any configured compound identifier that does not already exist to the azAEID database.
DoNotWriteExternalId
If specified, the value is used for lookups only. If the external compound identifier is not already in the database, it will not be added. A null enterprise identifier can result.
AllowNonTextCharacters
If not specified, an exception will be thrown when identifiers provided to ConfigureIdentifier contain non-text characters (line feeds, carriage returns, tabs, and control characters) Non-text characters can cause issues when database writes occur, so this option should only be used as a last resort.
CompoundIdDomainFirst
When generating the compound identifier from the ExternalId and AssigningDomain, put the domain first
AllowNullDomain
Allow identifiers with an empty domain, in which case the compound identifer will equal the ExternalId
