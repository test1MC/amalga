public enum EnterpriseIdResultType

Members

Member nameDescription
StandardMatchFKID
The EIDforXID value found by the MapSourceIdentifier(ISourceIdentifierRequest) method. For example, the MapSourceIdentifier(new PatientIdentifier(“MRN”, “Domain”)) method looks up the EIDforSOID value. The GetResult(EnterpriseIdType.SOID, EnterpriseIdResultType.StandardMatchEIDforXID) method call returns null if the MapSourceIdentifier(ISourceIdentifierRequest) method does not specify the same tier (SOID in the example).
CustomMatchFKID
The EIDforXID value found by the custom matching algorithm for the same tier. The GetResult method returns null if no custom match is performed or no match is found.
AnyFKID
Used to get a single, non-null EIDforXID if possible.
CurrentXID
The current XID as found or created by a standard lookup at this tier, inferred from a lower tier, or a custom match.
CustomMatchCurrentXID
The current XID as returned by the custom match.
Any
Returns the first non-null in order (standard match EIDforXID, custom match EIDforXID, current XID)
