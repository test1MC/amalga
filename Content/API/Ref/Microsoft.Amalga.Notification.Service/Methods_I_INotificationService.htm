INotificationService Methods
INotificationService Interface  See Also 

The INotificationService type exposes the following members.

Methods

  Name Description
AddNotificationTypes
Adds a set of notification type definition to the notification service store. This API supports initial application registration of the notification types it uses
DeleteNotificationTypes
Delete notification type definitions for indicated type ids.
DeleteNotificationTypesForApplication
Delete all the notification type definitions for the indicated application.
GetNotificationInfo
Supports Notification Manager fetching notification info on page load/reload
GetNotificationInfoForUser
Temporary API for testing end to end scenarios for PatientMatching etc. till the real impersonation through CreateChannelActingAs works. Or, if we decide that this is apropriate, this might stay
NotifyUser
A request to notify a user with the indicated notification type. This call creates a user notification request in the notification store.
NotifyUserWithResourceReferences
A request to notify a user with the indicated notification type. This call creates a user notification request in the notification store.
QueryNotificationOccurrences
To be supplied.
QueryNotificationTypes
Query for notification types with a filter
SaveNotifications
Saves notification occurrences in the notification store.
UpdateNotificationStates
Updates notification state for one or more notifications
UpdateNotificationTypes
Updates a set of notification type definition in the notification store. This API supports modifying an existing notification type.
