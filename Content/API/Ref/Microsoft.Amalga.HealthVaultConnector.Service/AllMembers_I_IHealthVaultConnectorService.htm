IHealthVaultConnectorService Members
IHealthVaultConnectorService Interface  Methods  See Also 

The IHealthVaultConnectorService type exposes the following members.

Methods

  Name Description
CreatePendingConnection
Method to create a pending connection with HealthVault by submitting a Direct-to-Clinical Request.
DeleteConnectionRequest
Method to delete a pending direct-to-clinical (D2C) connection
GetConnectionConfiguration
Method to obtain configuration needed for creating a network connection with HealthVault
GetHVConnections
Method to get all known HV connections for a patient. Connection may exist in any of the following states: Pending, Deleted, Authorized or UnAuthorized
SendItems
Method to send given list of HealthRecordItem to HealthVault. Based on FilterXpath provided, the item will either get added or updated in HealthVault.
SendVisitSummary
Method to create and send a visit summary in required format to HealthVault. HV Connector uses Clinical Document Application to generate the visit summary document File attachments associated with Visit summary will also be sent, if available.
SyncAuthorizedConnections
Method to synchronize authorized connections with Healthvault by reading new authorized connections since last sync.
UpdateAuthorizedRecordIds
Method to trigger an update of the internal HealthVault user-id to Amalga id mapping.
