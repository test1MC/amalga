IClinicalVisitSummaryDocumentBuilder..::.CreateDocument Method
IClinicalVisitSummaryDocumentBuilder Interface  See Also 

Creates a list of clinical documents in the specified formats constructed out of the patient data existing in the Amalga DB. All HealthVault documents returned in a single method call will have the same client-thing-id. Once uploaded to HealthVault, those things will be related. This method may also return a list of references to the related binary content. Relationship between the document and corresponding binary content is maintained by ClinicalDocument.Metadata.SetId value.
