IEventManagementService Methods
IEventManagementService Interface  See Also 

The IEventManagementService type exposes the following members.

Methods

  Name Description
ActivateEventType
Activates the specified events.
CreateEventType
Creates an Event with specified information and returns the event identifier
DeactivateEventType
Deactivates the specified events
DeleteEventType
Deletes the event with the specified Identifier
EnableEventingForApplication
Enables the change tracking for event by validating and enabling any underlying db requirements
FindEventMessages
Returns event data generated by the change tracker that matches the specified request.
FindEvents
Returns list of events created by the specified owner. In the first version, we require that the logged in user is same as the owner.
FindEventTypeForId
Returns an instance of the event id for the specified request criteria
Subscribe
Add new subscription to an existing event
Unsubscribe
Unsubscribe from an event
UpdateEventType
Updates the event type in the repository. This method will update the repository only if the resource already exists and will throw an exception if the specified event does not exist in the repository. This method assumes that the caller will set the Subscription list to NULL if it does not want to update any subscriptions. If subscription list is empty, then all the subscriptions for the event will be deleted. If subscription list is not empty, then all the existing ones will be deleted and the ones specified in the list will be inserted. Note that this method will also NOT update any LSN object references.
