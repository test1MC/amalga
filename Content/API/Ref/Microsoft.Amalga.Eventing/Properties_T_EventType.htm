The property collection that is used to connect to identify the resource. For Virtual Application view, the required properties to monitor an resource are: ApplicationViewName - The application view's name. The unique name for the application view. This information is passed to the IResourceResolver instance for the specified ResourceType. The properties and the key names will be different based on the resource type.
