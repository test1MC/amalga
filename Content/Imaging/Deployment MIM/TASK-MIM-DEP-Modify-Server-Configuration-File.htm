Server configuration file samples

 

Find the following samples for the server configuration files for the two topologies.

Sample Server Configuration File for Centralized Topology

<?xml version="1.0" encoding="utf-8"?>

<ImagingTopology xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns:xsd="http://www.w3.org/2001/XMLSchema" >

<!-- Imaging Data Server information (ServerName must be assigned the name of the server hosting DicomMetaData database)-->

<ImagingDataServer ServerName ="vm-imaging-db" UserId="az-sa" Password="passw0rd!" /-->

<!-- Site Name (choose a name for your centralized setup. This id will be used to find the IDS application.) -->

<SiteID Name="SingleSite" />

</ImagingTopology>

Sample Server Configuration File for Distributed Topology

<?xml version="1.0" encoding="utf-8"?>

<ImagingTopology xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" >

<!—IRDD Server information (ServerName should be assigned the name of the server hosting IRDD)-->

<ImagingDataServer ServerName ="vm-IRDD" UserId="az-sa" Password="passw0rd!" /-->

</ImagingTopology>
