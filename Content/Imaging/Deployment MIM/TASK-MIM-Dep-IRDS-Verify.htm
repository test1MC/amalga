Verify the application

 

The should be installed on a logical host at the central site.

To verify the deployment of the

  1. From Doors Start, click All Programs, point to , point to Visual Studio Tools, and then click Visual Studio Command Prompt.
  2. Type: WCFTestClient.
  3. Right-click My Service Project and then click Add service.
  4. Type: https://Your_Machine_FQDN:8801/Your_Instance_Name/Microsoft.Amalga.Imaging.Registry.ImagingRegistryService.svc

    in the Add service box

  5. Edit the configuration by changing MaxBufferSize and MaxReceivedMessageSize from the default to 1073741824.
  6. Type: 28cc695d-4891-4a47-827a-c7dcc3a92ef0 in the patient ID box.
  7. Click Invoke. Verify the presence of the study, series, and instance-level metadata.
