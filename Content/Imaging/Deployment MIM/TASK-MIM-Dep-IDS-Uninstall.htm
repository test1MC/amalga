Uninstall application

 

Please refer to the instructions to uninstall -hosted applications using . After completing the installation steps, remove the remaining service bits by running the following command.

.\SetupImageDeliveryRestService.ps1 -Uninstall -RestServiceInstanceName RestServiceInstanceName

Placeholder Value
RestServiceInstanceName The instance name for the respective service instance.

This script must be run on each node of the logical host or hosts that hosted one or more application instance.
