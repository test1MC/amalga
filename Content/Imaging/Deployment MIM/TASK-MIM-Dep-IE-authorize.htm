Authorize user roles to access entities

 

You can use the tool to authorize user roles to access entities.

To authorize a user role to access Imaging entities

  1. Open the module. For more information on tool see Open the module on page 1.

  2. Navigate to Amalga Entity Registry, click Imaging, click DICOMStudy, and click Access Policy.

  3. Select Authorize Role link.

  4. Grant Read access to the user role of your choice, for example, BUILTIN\AmalgaUsers.
