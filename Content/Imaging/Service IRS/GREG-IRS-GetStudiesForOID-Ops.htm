StudyInfoEntry[] GetStudiesForOID(string patientID)

The interface accepts an unique patient ID and returns an array containing study information. The unique patient ID in this case represents the internal ID of the patient.

 

Get studies for patient

 

The IRDS can provide study information using the OID.

The OID corresponds to a patient. The application passes OID and the IRS will return the studies belonging to the OID.
