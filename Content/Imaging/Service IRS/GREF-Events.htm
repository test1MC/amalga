Performance counters for the

 

This topic describes Performance counters for the .

The service () event uses the following performance counters.

Performance counter name Description
Query Requests Received Total The total number of query requests received by the IRS since the service was started.
Query Requests Received/second The rate of query requests received by the IRS.
Query Request Failure Total The total number of failed query requests by the IRS since the service was started.
Query Request Failure/second The rate of query request failure by the IRS.
Query Average Response Time The average response time for the IRS to handle a request.
