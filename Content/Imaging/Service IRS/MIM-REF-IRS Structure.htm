This topic introduces the structure and processes provided by the application ().

The service (IRS) is the only service hosted on the . The IRS provides a list of performed studies and associated metadata for a given patient, including the location of the corresponding BLOB for each study. A client application can then leverage the APIs provided by the service (IDS) for further BLOB-related requests, such as thumbnail generation.

APIs for the

GetDicomStudiesForPatient

GetDicomStudiesForPatient represents the query operations that IRS supports. By passing a patient ID to the interface, the IRS returns a list of the study information entries.

For more information, see Get studies for patient on page 1.

GetImageDeliveryApplicationInstanceName

You can use the to retrieve the configuration information of the IDS instance for a given site ID.

For more information, see Name resolution instance operation on page 1.
