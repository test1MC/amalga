Hide a column in the default list

Youcan hide the columns in the default list of fields by editing theDisplay columns property of the .

The default columns are:

  • StudyDatetime
  • NumberofSeries
  • Modality
  • Description
  • Institution
  • Account
  • AccessionNumber
  • ReferringPhysician
  • Department

To add a column in the default list

  1. Right-click the in , and then select Web Part Properties.
  2. Expand the Study Browser group.
  3. Add the column name to the Display columns box.
  4. Click OK.

To hide a column in the default list

  1. Right-click the in , and then select Web Part Properties.
  2. Expand the Study Browser group.
  3. Remove the column name from the Display columns box.
  4. Click OK.
