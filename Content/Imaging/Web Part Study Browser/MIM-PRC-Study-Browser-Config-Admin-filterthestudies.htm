Filter the studies by modality

You can filter thestudieslist by modalities.

To filter the studieslistby modality

  1. Click Edit Page. Right-click the, and select Web Part Properties.
  2. Expand the Study Browser group.
  3. Type the modalities, separated by semicolons, in the Modality Filter box; for example, CT;MN;US.
  4. Click OK.
