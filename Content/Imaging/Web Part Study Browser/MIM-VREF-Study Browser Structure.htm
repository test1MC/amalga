Data in the

The control displays the following data for each study in patient history.

Name Type or value Note
Date/Time mm/dd/yyy hh:mm The date and time of the study.
No. Series Number The count number of the series displayed in the control.
Modality Modality The abbreviation of each supported modality.
Description Text Descriptive text about the study.
Institution Text The name of the institution where the study was performed.
Account Number The patient account number.
Accession Number Number The accession numbers associated with radiology reports and studies.
Referring MD Text The name of the referring doctor.
Department Text The department name.

If the application fails to return data, thewill leave the corresponding cell blank. An error message will display the reason for the failure.
