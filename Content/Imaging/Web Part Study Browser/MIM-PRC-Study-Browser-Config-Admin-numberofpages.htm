Configure the list to set the number of studies to display on each page

You can configure the list to set the number of studies to display on each page.

To configure the list to set the number of studies to display on each page

  1. Right-click the in, andthenselect Web Part Properties.
  2. Expand the Study Browser group.
  3. Type the number in the Number of Studies per page box.

    The default value is “0”, which means no pagination. The will show all studies on one page

  4. Click OK.
