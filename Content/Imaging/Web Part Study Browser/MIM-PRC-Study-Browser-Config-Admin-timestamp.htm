Configure the list to sort byacolumn

You can configure the list to sort by a column.

To configure the list to sort by a column

  1. Click Edit Page. Right-click the, and then select Web Part Properties.
  2. Expand the Study Browser group.
  3. Type the name of the column in the Sorting by box.
    • To sort by descending,add "-" to the end of the column name; for example, columnname-.
    • To sort by ascending,add "+" to the end of the column name; for example, columnname+.
  4. Click OK.
