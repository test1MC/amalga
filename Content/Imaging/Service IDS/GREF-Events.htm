Event logs for the

 

This topic describes event logging for the  application ().

Performance markers measure the duration from the time a request is received until a response is sent. These markers are recorded in the performance logs after the service has been started.

You can use Event Tracing for Doors (ETW) for troubleshooting purposes. For more information, see "Improve Debugging and Performance Tuning with ETW."

Performance counters

  • Total number of requests rejected by .

  • Number of BLOB retrieval requests received per second.
  • Number of thumbnail retrieval requests received per second.
  • Number of thumbnail retrieval request failures per second.
  • Number of BLOB retrieval request failures per second.
  • Average time for the to handle BLOB retrieval requests.

    This time is defined as the interval between receipt of the request and the sending of the first byte to the client.

    This time is defined as the interval between receipt of the request and the sending of the first byte is sent to the client.
