Get-AmTrustedSTSCollection

 

The Get-AmTrustedSTSCollection cmdlet gets the token signing certificates of the Security Token Services (STS) that are registered with the Security Service. The token signing certificate of the Service is considered the primary STS instance and will always be returned by this cmdlet. If is installed, this cmdlet will return the token signing certificate of the Sharepoint STS as a non-primary STS instance.

Syntax

Get-AmTrustedSTSCollection

Parameter descriptions

This cmdlet does not have parameters.

Example

Get-AmTrustedSTSCollection
