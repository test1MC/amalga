Sync-AmManagementConsolePackage

 

Sync-AmManagementConsolePackage cmdlet brings down a management console package from the topology and applies it to a physical host.

Amalga built-in role membership required: AmalgaSysAdmin

For more information, see Built-in roles.

Syntax

Sync-AmManagementConsolePackage -Name <String> [-Force]

Parameter descriptions

-Name <String>

The name of the Management Console package as a string.

Characteristic Value
Required? Yes
Position Named
Default value None
Accepted values

The name of the Management Console package as a string.
Accept pipeline input? False
Accept wildcards? No

[-Force]

Applies the package to the physical host even if the package already exists.

Characteristic Value
Required? No
Position Named
Default value None
Accepted values Not applicable
Accept pipeline input? False
Accept wildcards? No
