Get-AmBuiltInRole

 

The Get-AmBuiltInRole cmdlet lists all built-in roles in an topology.

Amalga built-in role membership required: AmalgaAppAdmin, AmalgaDataAdmin, AmalgaDataPrivilegedService, AmalgaService, or AmalgaSysAdmin

For more information, see Built-in roles.

Syntax

Get-AmBuiltInRole

Parameter descriptions

This cmdlet does not have parameters.

Example

Get-AmBuiltInRole
