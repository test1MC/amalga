Remove-AmRoleFromAccessLevel

 

The Remove-AmRoleFromAccessLevel cmdlet removes permissions for application access levels from the specified roles.

Amalga built-in role membership required: AmalgaAppAdmin or AmalgaSysAdmin

For more information, see Built-in roles.

Syntax

Remove-AmRoleFromAccessLevel -ApplicationOrFeatureInstanceName <String> -AccessLevelName <String> -Roles <String>

Parameter descriptions

Example

Remove-AmRoleFromAccessLevel -ApplicationorFeatureInstanceName LabApp1 -AccessLevelName Writer -Roles DomainName\LabAdmin,DomainName\Manager
