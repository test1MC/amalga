Get-AmAccessLevel

 

The Get-AmAccessLevel cmdlet lists all access levels defined for an application.

Amalga built-in role membership required: AmalgaAppAdmin, AmalgaDataAdmin, AmalgaDataPrivilegedService, AmalgaService, or AmalgaSysAdmin

For more information, see Built-in roles.

Syntax

Get-AmAccessLevel -ApplicationOrFeatureInstanceName <String>

Parameter descriptions

Example

Get-AmAccessLevel -ApplicationorFeatureInstanceName LabApp1
