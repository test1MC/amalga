Update-AmSTSTokenSigningCertificate

 

Updates the non-primary Security Token Service (STS) token signing certificate.

Syntax

Update-AmSTSTokenSigningCertificate -Identifier <String> -TokenSigningCertificate <String>

Parameter descriptions

[-Identifier] <string>

Takes the identifier of the non-primary Security Token Service (STS).

Required? Yes
Position Named
Default value None
Accepted values

Identifier of the non-primary Security Token Service (STS).
Accept pipeline input? False
Accept wildcards? No

[-TokenSigningCertificate] <string>

Takes the token signing certificate of the non-primary Security Token Service (STS), encoded as a base64 string.

Required? Yes
Position Named
Default value None
Accepted values

Token signing certificate encoded as a base64 string. Example: $cert64
Accept pipeline input? False
Accept wildcards? No

Example

Update-AmSTSTokenSigningCertificate -Identifier Sharepoint -TokenSigningCertificate $cert64
