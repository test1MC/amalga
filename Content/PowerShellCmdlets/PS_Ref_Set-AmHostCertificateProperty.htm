Set-AmHostCertificateProperty

 

The Set-AmHostCertificateProperty cmdlet sets the SSL certificate on the logical host. This certificate will be used as the service and token encryption certificate for services or components that are assigned to this logical host.

Syntax

Set-AmHostCertificateProperty -Type <String> -InstanceName <String> -PropertyName <String> -PropertyValue <String> -PropertyType <String> -PropertyCategory <String> -OverwriteOption <String>

Parameter descriptions

[-Type] <string>

Takes the instance type of the item.

Required? Yes
Position Named
Default value None
Accepted values LogicalHost
Accept pipeline input? False
Accept wildcards? No

[-InstanceName] <string>

Takes the instance name of the logical host.

Required? Yes
Position Named
Default value None
Accepted values Name of the instance of the logical host.
Accept pipeline input? False
Accept wildcards? No

[-PropertyName] <string>

Takes the name of the property.

Required? Yes
Position Named
Default value None
Accepted values

HostCertificate

PlatformCertificate

AdminCertificate

Accept pipeline input? False
Accept wildcards? No

[-PropertyValue] <string>

Takes the thumbprint of the SSL certificate.

Required? Yes
Position Named
Default value None
Accepted values Thumbprint of the SSL certificate (can contain spaces).
Accept pipeline input? False
Accept wildcards? No

[-PropertyType] <string>

Takes the type of the property.

Required? Yes
Position Named
Default value None
Accepted values String
Accept pipeline input? False
Accept wildcards? No

[-PropertyCategory] <string>

Takes the category of the property.

Required? Yes
Position Named
Default value None
Accepted values Metadata
Accept pipeline input? False
Accept wildcards? No

[-OverwriteOption] <string>

Overwrites any existing SSL certificate for the specified logical host; takes no input.

Required? Yes
Position Named
Default value None
Accepted values Overwrite
Accept pipeline input? False
Accept wildcards? No

Examples

Set-AmHostCertificateProperty -Type LogicalHost -InstanceName ApplicationLogicalHost -PropertyName HostCertificate -PropertyValue "3f b0 6f 20 5b e4 7e 2f 30 70 65 c0 28 8b de 72 0a b8 9d c9" -PropertyType String -PropertyCategory Metadata -OverwriteOption Overwrite

Set-AmHostCertificateProperty -Type LogicalHost -InstanceName PlatformLogicalHost -PropertyName PlatformCertificate -PropertyValue 308efdee6453fff68c402e5eceee5b8bb9eaa619 -PropertyType String -PropertyCategory Metadata -OverwriteOption Overwrite

Set-AmHostCertificateProperty -Type LogicalHost -InstanceName AdministrationLogicalHost -PropertyName AdminCertificate -PropertyValue " 30 8e fd ee 64 53 ff f6 8c 40 2e 5e ce ee 5b 8b b9 ea a6 19" -PropertyType String -PropertyCategory Metadata -OverwriteOption Overwrite
