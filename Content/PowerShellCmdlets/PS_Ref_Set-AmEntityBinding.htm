Set-AmEntityBinding

 

Sets binding SQL statements for entities.

Amalga built-in role membership required: AmalgaDataAdmin, AmalgaSysAdmin

For more information, see Built-in roles.

Syntax

Set-AmEntityBinding -FilePath <String> [-Force]

Parameter descriptions

[-FilePath] <String>

Specifies the full path to a valid entity binding XML document containing binding SQL statements for indicated entities; see Entity binding XML overview.

Characteristic Value
Required? Yes
Position Named
Default value None
Accepted values A full path to a valid entity binding XML document. Examples: "c:\temp\mybindings.xml" or "\\testserver\temp\mybindings.xml".
Accept pipeline input? False
Accept wildcards? No

[-Force]

Overwrites any existing binding SQL statements for the specified entities; takes no input.

Characteristic Value
Required? No
Position Named
Default value None
Accepted values None
Accept pipeline input? False
Accept wildcards? No

Example

Set-AmEntityBinding -FilePath c:\temp\mybindings.xml -Force
