Bindings element

 

This topic describes the Bindings element.

Contains an ordered set of entity bindings.

Syntax

<Bindings>
   <Binding>...</Binding>
</Bindings>

Element characteristics

Characteristic Description
Data type and length None
Default value None
Cardinality 1-1: Required element that can occur once and only once.

Element relationships

Relationship Element
Parent elements None
Child elements Binding

Remarks

The Bindings element is the topmost element of an entity binding XML document, used with the Set-AmEntityBinding cmdlet to define entity bindings. The Bindings element contains an ordered collection of Binding elements, each of which represents an entity binding to be applied to a single entity.
