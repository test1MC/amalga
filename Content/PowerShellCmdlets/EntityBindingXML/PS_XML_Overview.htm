Entity binding XML overview

 

This topic provides an overview of the entity binding XML document.

The entity binding XML document contains a single Bindings element, which in turn contains an ordered collection of one or more Binding elements. Each Binding element represents a single entity and contains a Transact-SQL SELECT statement, called a binding SQL statement, used to select data for the entity identified by the Binding element. The Binding elements are processed in the order in which they are included in the Bindings element.

Example

The following example represents an entity binding XML document containing Binding elements for two entities, named Entity1 and Entity2, in the entity namespace. The Entity1 entity binding is configured for materialization.

<?xml version="1.0" encoding="utf-8" ?>
   <Bindings>  
      <Binding 
         EntityName='Entity1' 
         EntityNamespace='Common.Beta2'
         Materialize=true>
      <![CDATA[
         SELECT [EID]
            ,[FName]
            ,[LName]
            ,[SSN]
            ,[DOB]
            ,[DOB_o]
         FROM [AzADT].[dbo].[HIS601]
         ]]>
      </Binding>
      <Binding 
         EntityName='Entity2' 
         EntityNamespace='Common.Beta2'>
      <![CDATA[    
         SELECT [EID]
            ,[Account]
            ,[MRN]
            ,[InstitutionName]
            ,[SetID]
            ,[ExternalPatientID]
            ,[AlternatePatientID]
            ,[LName]
            ,[FName]
            ,[MName]
            ,[MotherMaidenName]
            ,[Sex]
            ,[PatientAlias]
            ,[Race]
            ,[Street1]
            ,[Street2]
            ,[City]
            ,[State]
            ,[ZipCode]
            ,[CountyCode]
            ,[CountryCode]
            ,[HPhone]
            ,[WPhone]
            ,[Language]
            ,[MaritalStatus]
            ,[Religion]
            ,[SSN]
            ,[DriverLicNum]
            ,[MotherID]
            ,[EthnicGroup]
            ,[BirthPlace]
            ,[MultipleBirthIndicator]
            ,[BirthOrder]
            ,[Citizenship]
            ,[VeteransMilitaryStatus]
            ,[DistrictCode]
            ,[DOB]
            ,[DOB_o]
         FROM [AzADT].[dbo].[PID601]
      ]]>
   </Binding>  
</Bindings>
