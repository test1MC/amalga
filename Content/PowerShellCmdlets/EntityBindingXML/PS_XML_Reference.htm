Entity binding XML reference

 

This topic describes the entity binding XML document, used by the Set-AmEntityBinding cmdlet.

The entity binding XML document is used by the Set-AmEntityBinding cmdlet to define entity bindings. Entity bindings link entities defined in the to the databases, tables, and columns in the storage schema.

You can also use to bind entities and to view currently applied binding SQL statements. For more information, see Entity binding management.
