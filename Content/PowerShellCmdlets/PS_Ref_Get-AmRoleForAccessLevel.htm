Get-AmRoleForAccessLevel

 

The Get-AmRoleForAccessLevel cmdlet lists all roles that are granted permissions to an application and access level.

Amalga built-in role membership required: AmalgaAppAdmin, AmalgaDataAdmin, AmalgaDataPrivilegedService, AmalgaService, or AmalgaSysAdmin

For more information, see Built-in roles.

Syntax

Get-AmRoleForAccessLevel -ApplicationOrFeatureInstanceName <String> -AccessLevelName <String>

Parameter descriptions

Example

Get-AmRoleForAccessLevel -ApplicationorFeatureInstanceName LabApp1 -AccessLevelName Writer
