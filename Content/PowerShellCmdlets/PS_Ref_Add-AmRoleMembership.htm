Add-AmRoleMembership

 

The Add-AmRoleMembership cmdlet assigns a user, group, or computer account to an role.

Amalga built-in role membership required: AmalgaSysAdmin

For more information, see Built-in roles.

Syntax

Add-AmRoleMembership -RoleName <String> -UserAccount <String> -DirectoryService <String>

Parameter descriptions

[-DirectoryService] <string>

Takes the name of a directory service provider supported by .

Required? Yes
Position Named
Default value None
Accepted values Master, Local
Accept pipeline input? False
Accept wildcards? No

Example

Add-AmRoleMembership -RoleName ERStaff -UserAccount DomainName\ERDoctors MASTER
