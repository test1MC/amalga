Update-AmRole

 

The Update-AmRole cmdlet revises the user-friendly display name for an role.

Amalga built-in role membership required: AmalgaSysAdmin

For more information, see Built-in roles.

Syntax

Update-AmRole -RoleName <String> -NewDisplayName <String>

Parameter descriptions

Example

Update-AmRole -RoleName ELNurses -NewDisplayName EastlakeNurses
