System module primary window

 

This topic displays an image of the System module primary window and describes its key elements.

 Review and manage the elements of the topology, including features, logical hosts, physical hosts, and data hosts.

Access For access steps, see Open the System module.

The following image shows the System module primary window.

Ref# Name Description and tasks
1 Navigation pane

Contains the list of installed features, logical hosts, physical hosts, and data hosts.
2 Title bar Indicates the active module.
3 Results pane Displays the details of the feature, logical host, physical host, or data host, selected in the navigation pane.
4 Tasks pane Displays the tasks available for the item selected in the navigation pane.
