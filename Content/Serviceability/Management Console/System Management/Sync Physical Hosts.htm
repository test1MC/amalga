Synchronize physical hosts

 

Synchronizing physical hosts aligns the state of the hosted services on a physical host with their definition in the Amalga topology (AmalgaSystem) database. You can synchronize physical hosts using the Sync-AmPhysicalHost cmdlet. For more information on this cmdlet, see Sync-AmPhysicalHost.

To synchronize a physical host.

  1. Log on to the physical host you want to synchronize.

  2. Start Doors PowerShell as an administrator. Click Start, click All Programs, click Accessories, click Doors PowerShell, and then right-click Doors PowerShell and click Run as Administrator.
  3. Load Microsoft.Amalga.Powershell.dll.

    1. Navigate to the appropriate folder.

      The default installation folder is: C:\Program Files\Microsoft Amalga\System 3.0\Platform\PowerShell.

    2. Enter the following cmdlet.

      Import-Module .\Microsoft.Amalga.Powershell.dll

  4. Enter the following cmdlet.

    Sync-PhysicalHost
