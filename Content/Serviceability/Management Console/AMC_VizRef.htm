Ref# Name Description and tasks Page
1 Title bar Identifies the selected module and provides additional options. For example, indicates selection of the System module with the label System Configuration.  
2 Tasks pane Lists tasks relevant to the selected item in the navigation pane.  
3 Details pane Lists information about the selected item in the navigation pane. For example, if a logical host is selected in the navigation pane, lists host type, host port number, and other details.  
4 Navigation pane Lists items relevant to the selected module. For example, if the System module is selected, lists features, logical hosts, and data hosts.  
5 Module list in the navigation pane Lists available modules. For example, Application, System, and EntityManagement.
