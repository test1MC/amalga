Computer accounts

 

This topic discusses computer accounts as related to roles.

In addition to user and group accounts, you can add computer accounts to roles.

For example, you may want a computer to access data and applications offline, without any user context at run time, to create reports or send scheduled alerts. Computer accounts can be managed like user and group accounts and you can assign them to dedicated custom roles.

By default, computer accounts for computers on which platform services or application services are installed are automatically assigned to the BUILTIN\AmalgaSysAdmin role.
