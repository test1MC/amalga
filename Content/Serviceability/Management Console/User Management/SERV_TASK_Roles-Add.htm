Add a role

 

This topic describes how to add roles to an topology.

You can add roles to an topology.

For example, you can add roles for physicians, emergency room staff, and nurses in each geographical location served by your site. If Dr. Smith is a physician at the Contoso East location as well as part of the emergency room staff, you can create a role to capture each unique set of access rights and assign Dr. Smith's user account to both roles.

To add an role to an topology, use the following cmdlet.

Add-AmRole

For instructions on accessing cmdlets, see PowerShell reference.
