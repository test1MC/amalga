Remove a role

 

This topic describes how to remove roles from an topology.

You can remove custom roles from an topology.

Built-in roles cannot be removed.

To remove an role, use the following cmdlet.

Remove-AmRole

For instructions on accessing cmdlets, see PowerShell reference.
