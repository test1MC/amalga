Remove accounts from a role

 

This topic describes how to remove assignment of accounts to roles.

You can remove previously assigned user, group, and computer accounts from roles.

To remove an account from an role, use the following cmdlets.

To remove this type of account... Use this cmdlet:
User, security group, or computer account Remove-AmRoleMembership
The current computer account Remove-AmMachineServiceAccount

For instructions on accessing cmdlets, see PowerShell reference.
