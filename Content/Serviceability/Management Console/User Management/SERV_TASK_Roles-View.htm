View roles and assigned accounts

 

This topic describes how to view roles and the accounts assigned to them.

You can view roles in the topology along with the user, group, and computer accounts assigned to them.

To view roles and assigned accounts, use the following cmdlets.

To view... Use this cmdlet:
All roles in a topology Get-AmRole
Built-in roles in a topology Get-AmBuiltInRole
Accounts assigned to a role Get-AmRoleMembership
Role membership for an account Get-AmUserRole

For instructions on accessing cmdlets, see PowerShell reference.
