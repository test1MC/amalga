Ref# Name Description and tasks
1 Search Results

Lists the semantic tags that match the specified search parameters. Results are listed with the following columns.

For information about semantic tags as stored in the terminology namespace, see Microsoft.Terminology.Concept entity.

2 Search Tags

Allows you to specify search parameters; you can select to find text in the tag description ("Meaning" property [attribute] of the concept) or to find text in the tag code ("Code" property [attribute] of the concept).

Regular expression strings are not supported for semantic tag search.

3 Remove Removes the semantic tag selected in the Selected Tags list.
4 Add

Adds the semantic tag selected in the Search Results list to the Selected Tags list.

5 Selected Tags Lists the semantic tags to be applied to the current entity or entity property; includes currently applied tags.
6 Search Attempts to find semantic tags that match the specified search parameters. The ENTER key is an alternative to this button.
