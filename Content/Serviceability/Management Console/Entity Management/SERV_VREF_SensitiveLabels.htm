 View the sensitive labels available for the current entity (includes those created for related entities), create sensitive labels for the current entity, and bind sensitive labels to properties of the selected entity.

Access For access steps, see Restrict access to an entity property (sensitive labels).

The following is an image of the pane for the Result entity, showing a sensitive label DiagnosisAbn that is applied to the properties AbnormalFlags and AbnormalFlagsNormalized.

Ref# Name Description and tasks
1 Sensitive Labels Lists the sensitive labels available for application to properties in the current entity, as well as the entity (Origin) and namespace (Schema) in which each sensitive label was created.
2 Sensitive Label Lists available sensitive labels for applying to the selected property.
3 Property Name Lists properties selected for applying sensitive labels.
4 Properties Lists the properties of the current entity and identifies default data types.
