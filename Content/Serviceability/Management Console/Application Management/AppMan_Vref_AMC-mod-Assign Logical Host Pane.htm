Assign logical host pane

 

This topic displays an image of the Assign Logical Hosts pane as seen during application deployment.

 Assigns services to logical hosts for deployment.

Access For access steps, see Assign application logical hosts.

The following image shows the Assign Logical Hosts pane during an application deployment

Ref# Name Description and tasks
1 Service assignment box

Contains one service of the application that needs logical host assignment. Only one is shown here, but there can be multiple service assignment boxes depending on the number and type of services associated with the application.
2 Service cluster mode Specifies the type of logical host environment required by the application service. In this case, Network Load Balancing.
3 Logical hosts Displays a list of logical hosts that will meet the cluster mode requirements of the application service. In this case, only the logical host named "FABLN" is shown as being available for assignment.
