Sync application pane

 

This topic displays an image of the Sync Application pane as seen during application deployment.

 Synchronizes the application to physical hosts.

Access For access steps, see AppMan Proc Application Sync.

The following image shows the Sync Application pane during an application deployment.

Ref# Name Description and tasks
1 Logical host name

Contains the logical host assigned to this application.
2 Physical hosts Contains the physical hosts comprising the logical host assigned to this application. Synchronization is targeted to physical hosts within the specified logical host.
3 Force check box Controls whether synchronization is forced on the physical hosts even if the application has been synchronized previously.
4 Sync Selected Initiates the synchronization process on the physical hosts selected.
