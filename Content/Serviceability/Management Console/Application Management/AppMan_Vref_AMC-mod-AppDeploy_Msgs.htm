Application status

Ref# Name Description and tasks
1 Navigation pane

Contains the list of installed applications.

2 Title bar Indicates the active module.
3 Results pane Displays the details of the application being deployed. NotesApp in this case.
4 Notification area Displays status messages about the application deployment.
5 Tasks pane Displays the tasks available for the item selected in the navigation pane.
