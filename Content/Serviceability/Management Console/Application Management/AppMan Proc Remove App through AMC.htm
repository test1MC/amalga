Remove an application with

 

Removing an application requires that you unsync and then unpublish the application.


Required Role: AmalgaSysAdmin, or AmalgaAppAdmin

To remove an application from Amalga

  2. In the navigation pane, select the application instance you want to remove.

  3. Unsync the application.

    1. In the tasks pane, click Unsync.
    2. In the Unsync Application pane, select the physical hosts to unsync.
    3. Click Unsync Selected.

    4. After the unsync operation is complete, click Close.

    After the unsync operation, the notification area of the results pane will display a host assignment error because the application is no longer synchronized to a physical host. The notification can be ignored.

  4. Unpublish the application.
    1. With the application still selected in the navigation pane, click Unpublish in the tasks pane.
    2. In the Unpublish Application pane, confirm the operation by clicking Unpublish.
