Publish an application

 

You can publish an application to the topology by using the Publish-ApplicationInstance cmdlet. For more information about the syntax and parameters of this command, see Publish-ApplicationInstance.

For information on how to use this cmdlet to install applications, see Install applications in the Installation Guide.

To publish an application

  1. Start as an administrator. Click Start, click All Programs, click Accessories, click Doors PowerShell, and then right-click Doors PowerShell and click Run as Administrator.

  2. Run the following commands to prepare .

    cd 'C:\Program Files\Microsoft Amalga\System 3.0\Platform\PowerShell'

    Import-module .\Microsoft.Amalga.Deployment.RuntimeLib.dll

  3. Run the following command to publish an application. Replace the placeholders with variables appropriate for your installation.

    Publish-ApplicationInstance -ApplicationCabFile "ApplicationCabPath" -ApplicationAuxiliaryFiles "AppManifestPath; ServiceManifestPath1; ServiceManifestPath2"
