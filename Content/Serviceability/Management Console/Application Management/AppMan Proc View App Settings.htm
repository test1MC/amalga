View application settings

 

For CTP, you can view application settings by using the Get-AmConfigurationSetting cmdlet. For more information on the syntax and parameters of this command, see Get-AmConfigurationSetting.

To view application settings

  1. Start Doors PowerShell as an administrator. Click Start, click All Programs, click Accessories, click Doors PowerShell, and then right-click Doors PowerShell and click Run as Administrator.

  2. Run the following commands to prepare .

    cd 'C:\Program Files\Microsoft Amalga\System 3.0\Platform\PowerShell'

    Import-module .\Microsoft.Amalga.Deployment.RuntimeLib.dll

    Import-Module .\Microsoft.Amalga.Deployment.TopologyLib.dll

  3. Run the following command to view application settings. Replace the placeholders with variables appropriate for your installation.

    Get-AmConfigurationSetting -InstanceType <ConfigurationSettingInstanceType> -InstanceName <InstanceName> [-SettingName <SettingName>]
