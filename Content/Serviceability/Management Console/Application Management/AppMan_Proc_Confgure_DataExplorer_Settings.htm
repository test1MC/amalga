Configure Settings

 

This topic describes how to configure settings for .

Several settings control the functionality of . These settings are set using the .

To configure settings

  2. In the module navigation tree, expand the Hosted Applications folder and click DataExplorerPrivate.

  3. In the tasks pane, click Configure Settings.

  4. In the Configure Application Settings pane, configure the following settings:

  • ListCacheTimeoutSeconds - enter an integer value for the number of seconds that the current cached list will remain accessible.

  • ListMaxRows - enter an integer value for the maximum number of rows to be included in the list

    A zero or negative value will cause a warning and the value will be set to 5000.

  • ListPageSize - enter an integer value for the number of rows to be displayed on a page.

    A zero or negative value will disable paging.

  1. After you are done configuring the settings, click Save.
