Deploy applications

 

This section introduces the sections necessary to deploy hosted applications either through the , or cmdlets.

hosted applications can be deployed to a topology through either the or cmdlets.

Using the provides a graphical user interface and a set of guided steps leading the administrator through application deployment. This method is most appropriate for occasional deployments, or ones involving few application instances.

Using cmdlets provides a command line experience that is best for repeated deployments, or ones involving multiple application instances.
