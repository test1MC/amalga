Access Inheritance dialog box

 

This topic displays an image of the Access Inheritance dialog box and describes its key elements.

 Lists application-private database tables for the selected application instance along with any public entities suggested for access inheritance.

Access For access steps, see Configure access inheritance for an application's private database tables.

The following image shows the Access Inheritance dialog box for the application PatientNotes_1, listing the application's private database tables (such as PatientNotes) along with public entities (such as Common.Beta2.Patient) that are suggested for access inheritance.

Ref# Name Description and tasks
1 Public Entity Suggested for Access Inheritance Lists public, read-only entities suggested in the application instance's virtual schema for access inheritance.
2 Number of Roles Authorized Indicates the number of roles that are authorized for the entity.
3 Save Saves changes and closes the dialog box.
4 (check box for table) Check box for an application-private database table. Select to inherit the access policy from the indicated public entity; clear to opt out of access inheritance.
5 Private Table Lists the private database tables for this application.
