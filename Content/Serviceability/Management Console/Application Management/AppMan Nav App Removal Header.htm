Remove applications

 

This section introduces the sections necessary to remove hosted applications from the topology either through the , or cmdlets.

hosted applications can be removed from a topology through either the or cmdlets.

Using the provides a graphical user interface and a set of guided steps leading the administrator through application removal. This method is most appropriate for occasional removals, or ones involving few application instances.

Using cmdlets provides a command line alternative that is better suited for scripted removals, or ones involving multiple application instances.
