Change application settings

 

For , you can change application settings by using the Set-AmConfigurationSetting cmdlet. For more information about the syntax and parameters of this command, see Set-AmConfigurationSetting.

To change application settings

  1. Start as an administrator. Click Start, click All Programs, click Accessories, click Doors PowerShell, and then right-click Doors PowerShell and click Run as Administrator.

  2. Run the following commands to prepare .

    cd 'C:\Program Files\Microsoft Amalga\System 3.0\Platform\PowerShell'

    Import-module .\Microsoft.Amalga.Deployment.RuntimeLib.dll

    Import-Module .\Microsoft.Amalga.Deployment.TopologyLib.dll

  3. Run the following command to change application settings. Replace the placeholders with variables appropriate for your installation.

    Set-AmConfigurationSetting –InstanceType <String> -InstanceName <String> -SettingName <String> [-SettingValue] <String> [-SettingValueFile] <String>
