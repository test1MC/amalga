Diagnostic application logs query

 

All log entries, for all time, for a given application, are sorted by date. Replace AuthorizationManagement with the name of the application desired. 

SELECT [Id]
      ,[EntryID]
      ,[TimeStampUTC]
      ,[ActivityID]
      ,[PId]
      ,[TId]
      ,[Level]
      ,[ServerName]
      ,[LogicalHostName]
      ,[ParentInstance]
      ,[ComponentInstance]
      ,[Source]
      ,[Context]
      ,[Message]
      ,[ExceptionName]
      ,[AssemblyName]
      ,[ClassName]
      ,[MethodName]
      ,[FileName]
      ,[LineNumber]
      ,[StackTrace]
  FROM [AmalgaSystem].[DiagLog].[DiagLogEntry]
  WHERE ComponentInstance = 'AuthorizationManagement'
  ORDER BY TimeStampUTC
