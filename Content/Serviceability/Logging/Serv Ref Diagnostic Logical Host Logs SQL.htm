Diagnostic logical host log query

 

All log entries, from all time, from the specified logical host, are sorted by date. Replace FabricLogicalHost with the appropriate logical host name. 

SELECT [Id]
      ,[EntryID]
      ,[TimeStampUTC]
      ,[ActivityID]
      ,[PId]
      ,[TId]
      ,[Level]
      ,[ServerName]
      ,[LogicalHostName]
      ,[ParentInstance]
      ,[ComponentInstance]
      ,[Source]
      ,[Context]
      ,[Message]
      ,[ExceptionName]
      ,[AssemblyName]
      ,[ClassName]
      ,[MethodName]
      ,[FileName]
      ,[LineNumber]
      ,[StackTrace]
  FROM [AmalgaSystem].[DiagLog].[DiagLogEntry]
  WHERE LogicalHostName = 'FabricLogicalHost' 
  ORDER BY TimeStampUTC
