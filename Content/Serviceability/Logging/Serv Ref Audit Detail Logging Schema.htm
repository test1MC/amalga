Audit detail table

 

Auditing data are stored in the AmalgaAudit database in two tables. Following are the column names, null status, and description of the Audit.AuditDetail table. For the higher level auditing table, see Audit table

Column name Nullable Description
AuditId False The unique identifier of the audit entry. AuditID plus sequence is the primary key of the AuditDetails table.
Sequence False The record sequence under the same auditing operation. AuditID plus sequence is the primary key of the AuditDetails table.
GroupIndex False The index of the group.
PropertyName False The name of the property.
Value False The value of the property.
PropertyType False The type of the propery being audited.
  • Entity property (EP)
  • Data connectory query (QU)
  • Others (OT)
IsFromTrustedSource False A binary field that specifies whether the audited operation originated from a trusted source.
IsFromPlatformService False A binary field that specifies whether the audited operation originated from a platform service.
