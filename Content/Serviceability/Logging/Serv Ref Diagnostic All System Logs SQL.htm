Diagnostic all system logs query

 

All log entries, from all time, for all of Amalga, are sorted by date. 

SELECT [Id]
      ,[EntryID]
      ,[TimeStampUTC]
      ,[ActivityID]
      ,[PId]
      ,[TId]
      ,[Level]
      ,[ServerName]
      ,[LogicalHostName]
      ,[ParentInstance]
      ,[ComponentInstance]
      ,[Source]
      ,[Context]
      ,[Message]
      ,[ExceptionName]
      ,[AssemblyName]
      ,[ClassName]
      ,[MethodName]
      ,[FileName]
      ,[LineNumber]
      ,[StackTrace]
  FROM [AmalgaSystem].[DiagLog].[DiagLogEntry]
  ORDER BY TimeStampUTC
