Diagnostic server logs query

 

All log entries from the specified server are sorted by date. Replace DevServer3 with the appropriate server name.

SELECT [Id]
      ,[EntryID]
      ,[TimeStampUTC]
      ,[ActivityID]
      ,[PId]
      ,[TId]
      ,[Level]
      ,[ServerName]
      ,[LogicalHostName]
      ,[ParentInstance]
      ,[ComponentInstance]
      ,[Source]
      ,[Context]
      ,[Message]
      ,[ExceptionName]
      ,[AssemblyName]
      ,[ClassName]
      ,[MethodName]
      ,[FileName]
      ,[LineNumber]
      ,[StackTrace]
  FROM [AmalgaSystem].[DiagLog].[DiagLogEntry]
  WHERE ServerName = 'DevServer3'
  ORDER BY TimeStampUTC
