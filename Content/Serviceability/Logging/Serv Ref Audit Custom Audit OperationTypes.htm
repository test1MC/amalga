Custom audit operation types

 

Developers can add calls to the custom audit application programming interface (API) to create supplemental audit records. Custom audit operation types allow applications running on the platform to add additional audit information in the context of the running application. The custom audit operations respect a set of predefined audit operation types. The types are described in the following table and contained in the Microsoft.Amalga.Hosting.Service.OperationType enumeration.

OperationType enumeration members

Member name Description
ViewData View data
UpdateData Update data
DeleteData Delete data
AddData Add data
ServiceRequest Service Request
Print Print
Download Download
Email Email
Fax Fax
Schedule Schedule a job
ExecuteSchedule Execute a scheduled job
Login User login
Logout User logout
