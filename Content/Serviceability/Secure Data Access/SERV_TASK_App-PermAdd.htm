Grant and remove permissions to an application

 

This topic describes how to grant and remove permissions for an role to use an application.

You can grant and remove permissions for an role to use an application instance at a particular access level.


Required Role: AmalgaAppAdmin or AmalgaSysAdmin

To grant or remove permissions for an application

  4. Click Edit.

    The Edit Permissions dialog box appears, listing access levels.

  5. Select the access level you want.

    All roles configured for the installation are listed.

  6. Do one of the following: 
    • To add a role to the selected access level (grant permissions), select the check box for that role.
    • To remove a role from the selected access level), clear the check box for that role.
  7. Click Save to save changes and close the Edit Permissions dialog box.
  8. Click Ok to close the Permissions dialog box and redisplay the application summary pane.
