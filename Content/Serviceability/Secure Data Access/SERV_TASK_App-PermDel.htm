Remove permissions for an application

 

This topic describes how to remove an role's permissions for application access levels.

You can remove an role's permissions for access levels of an application instance.


Required Role: AmalgaAppAdmin or AmalgaSysAdmin

To remove permissions for an application

  4. Click Edit.

    The Edit Permissions dialog box appears, listing access levels.

  5. Select the access level you want and then clear the check box for the role that you want to remove from the access level.
  6. Click Save to save changes and close the Edit Permissions dialog box.
  7. Click OK to close the Permissions dialog box and redisplay the application summary pane.
